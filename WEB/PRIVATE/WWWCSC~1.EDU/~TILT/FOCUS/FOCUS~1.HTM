FOCUS -- A FOrm Customizable Shell

Note: The distribution, focus.tar.Z v1.0, is available, as some may find the code useful. The code is only known to work on a sun4 system, however -- caveat emptor! I don't currently have it available via FTP, so if you're interested send me e-mail. Feel free to attempt to modify it to work on other systems. The documentation is also available in HTML format.

FOCUS is a curses based form processor, which provides
  • The ability to fill out forms and send via e-mail
  • Functionality as a nice "front end" for other unix programs
  • Ability to provide a "hypertext-like" menuing interface for unix
FOCUS began life as an Inter-Library Loan form processor.

It originally provided simply for text fields which could be filled out, and the ability to mail a form to a particular person. It has since gained the following "features":

  • Editable Text Fields
  • Toggle Fields with Multiple Preset Values
  • Buttons which can perform actions (such as calling another program, or calling another form)
