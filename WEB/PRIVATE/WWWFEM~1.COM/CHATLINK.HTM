Cygnus Web-Chat Live - The first chat-server I ever went to and in many ways it has changed my life! I wish I could provide a link for it but the sad truth is it's gone.. for now.. Cygnus will return!


Novia Chat - A good chat server that doesn't require for you to have the newest version of Netscape like most of the other chat servers do.


Download PowWow - This is one of my favourite ways of chatting. Just download the software and then you can try paging me but you must have PowWow running to do this.


Forum Chat - A Danish chat that is often busy in the afternoon and in the evening. There's only room for 10 people in their public room but you're welcome to try and see if you're lucky.


Fire Fly Chat - The best chat server I've seen since Cygnus shut down. Includes some great features like a mail system for the users. I really like this place!


Download mIRC - A program for using Internet Relay Chat. The original, text-based way of chatting on the net but unfortunately, it hasn't changed much since it started.


Mødestedet - A Danish chat with unlimited number of people and where the server runs pretty well. But I still haven't had a serious discussion there yet...
