Netscape Navigator - The best browser available. Make sure to download the new version for your viewing pleasure. A lot of pages have started containing things like Java so you should make sure to have a browser that supports it.


Internet Explorer - Microsofts web browser. Personally, I prefer Netscape but you're free to choose your favourite browser. Also, Internet Explorer supports some new features.


Tucows Website - Tons of great software (Paint Shop Pro, CuteFTP etc.) and a *must* for your bookmarks.


HotDog Pro - My favourite web editor that I have used to make these pages. Don't get scared by the name; it's a pretty serious program! Download a free shareware version.


Dogz Adopt Me - A cute little dog that can run on your desktop. If you've always wanted a dog but couldn't have one this is the solution.


Catz Adopt Me - Like Dogz although this time it's a cute little kitten in stead. I haven't tried it yet, though, so please let me know if it's good by e-mailing me.
