Setup:

Starting Point is designed to make your daily Web experience more productive. Starting Point helps you manage Web resources by combining a powerful, centralized Web searching tool with quality, subject-oriented Web resources you need every day -- all combined in an easy-to-use, comprehensive, and well organized starting point.

We recommend setting your browser start-up page to Starting Point. This enables you to access Starting Point automatically when you load your browser, or by simpling clicking your browser's 'Home' icon. Here's how:

  • Set Starting Point as your start-up page
  • Add Starting Point to your bookmarks / hotlist
  • Create a desktop shortcut to Starting Point (Win95)
  • Add Starting Point HTML code to your own Web page

Here's how to set Starting Point as
your start-up page:
  • Netscape 3.0
    1. Click "Options" on menu bar
    2. Click "General Preferences..."
    3. Select "Appearance" tab
    4. In the "Startup" section, click the "Home Page Location" button
    5. In the "Startup" section, enter "http://www.stpt.com" in the text-entry box.

  • Netscape 2.0
    1. Click "Options" on menu bar
    2. Click "General Preferences..."
    3. Select "Appearance" tab
    4. Type "www.stpt.com" as your "Start With: Home Page Location"

  • Netscape 1.2 and previous versions
    1. Click menu bar "Options"
    2. Click "Preferences..."
    3. Select pull-down menu "Styles"
    4. Type "www.stpt.com" as your "Start With: Home Page Location"
    5. Add Starting Point to your bookmarks by clicking on menu bar "Bookmarks," then click "Add Bookmark."

  • Microsoft Internet Explorer
    1. Go to Starting Point start page
    2. Click "View" on menu bar
    3. Click "Options"
    4. Select "Start and Search Pages" tab
    5. Click button "Use Current" then click "Apply"
    6. You may do this for both Start and Search Pages

  • NetCruiser
    1. Click menu bar "Settings"
    2. Click "WWW Options"
    3. Type "http://www.stpt.com/" as your "Home Page:"

  • AOL Web Browser:
    1. Add Starting Point as your start-up page by clicking "Prefs"
    2. Change Home Page location to "http://www.stpt.com"

  • WebExplorer (OS2 Warp)
    1. Click menu bar "Configure"
    2. Click "Servers"
    3. Type "http://www.stpt.com/" as your "Home Document URL"
    4. Click "Load at startup?" checkbox

  • NCSA Mosaic
    1. Click menu bar "Options"
    2. Click "Preferences..."
    3. Select "Document" tab
    4. Type "http://www.stpt.com/" as your "Home Page"

  • Spry Mosaic
    1. Click menu bar "Options"
    2. Click "Configuration"
    3. Type "http://www.stpt.com/" at the "URL:" prompt

Here's how to add Starting Point
to your bookmarks / hotlist:

  • Netscape
    1. Click "Bookmarks" on menu bar
    2. Click "Add Bookmark"

  • Microsoft Internet Explorer:
    1. Click "Favorites" on menu bar
    2. Click "Add To Favorites"

  • NetCruiser
    1. Click "Bookmarks" on menu bar
    2. Click "Add Bookmark"

  • AOL Web Browser
    1. Click "Favorite Places" on tool bar
    2. Click "ADD FAVORITE PLACE" button
    3. Type "Starting Point"
    4. Type "www.stpt.com"
    5. Click "CLOSE" button

  • Mosaic
    1. Click "Hotlist" on menu bar
    2. Click "Add to Hotlist"

  • WinWeb
    1. Click "Bookmarks" on menu bar
    2. Click "Add Bookmark"

  • Prodigy Web Browser
    1. Click "HOTLIST" button
    2. Click "ADD TO HOTLIST"

Here's how to create a desktop shortcut to Starting Point using Windows 95:
  • Netscape
    1. Place your mouse pointer directly over this Starting Point link
    2. Hold down your left mouse button
    3. Drag to your Windows 95 desk top
    4. Release your left mouse button

  • Microsoft Internet Explorer
    1. Place your mouse pointer directly over this Starting Point link
    2. Hold down your left mouse button
    3. Drag to your Windows 95 desktop
    4. Release your left mouse button

Here's how to add Starting Point HTML code to your own Web page:

You may copy and paste the following HTML code into your Web page.

HTML code for a text link:

<!-- begin text link -->
<a href="http://www.stpt.com/">Starting Point</A>
<!-- end text link -->

HTML code for a graphical link:

<!-- begin graphical link -->
<a href="http://www.stpt.com/">
<img src="http://www.stpt.com/stpthalf.gif"
border=0 width=139 height=43 alt="Starting Point"></a>
<!-- end graphical link -->
