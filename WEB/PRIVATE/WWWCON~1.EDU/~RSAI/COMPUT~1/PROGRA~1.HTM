Interface Technologies Online Training Center
Online programming tutorials on a variety of topics including Visual C/C++, ISAPI extensions, Java, web development, as well as numerous other topics.

IPng: IP Next Generation
IPng is a new version of IP which is designed to be an evolutionary step from IPv4 and is interoperable with the current IPv4. IPng is designed to run well on high performance networks (e.g. ATM) and at the same time is still efficient for low bandwidth networks (e.g. wireless).

Stardust WinSock Labs
The definitive independent hub of information and services for Windows software developers creating TCP/IP-based Internet and Intranet applications for Microsoft Windows.
