 		How our site is produced

NetGuide Live is based on a high-performance technology platform. The system has been designed to be robust, scalable, and open. A modular, object-oriented architecture improves efficiency and allows for easy integration of new functionality and information sources and powerful third-party technologies.

The latest dynamic Web technologies, such as Java, Shockwave, and ActiveX, will be used to create exciting and intuitive interfaces for the user. In addition, we have developed an extended HTML authoring environment that allows designers to create and manage the components that are used to dynamically generate Web pages.

For our production processes, we have developed a powerful editorial workflow management and production system, including agents that traverse the Web and sophisticated text analysis facilities.
