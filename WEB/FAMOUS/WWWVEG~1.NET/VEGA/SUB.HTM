All routine administrative requests (including subscriptions and unsubscriptions) concerning this mailing list are handled by an automated server. Please read this message carefully to find the information relevant to you.

SUBSCRIBING

To subscribe to undertow, send the following in the body (not the subject line) of an email message to Majordomo@majordomo.law.emory.edu

subscribe undertow

This will subscribe the account from which you send the message to the undertow list.

If you wish to subscribe another address instead (such as a local redistribution list), you can use a command of the form:

subscribe undertow other-address@your_site.your_net

UNSUBSCRIBING

To unsubscribe from undertow, send the following in the body (not the subject line) of an email message to Majordomo@majordomo.law.emory.edu .

unsubscribe undertow

This will unsubscribe the account from which you send the message. If you are subscribed with some other address, you'll have to send a command of the following form instead:

unsubscribe undertow other-address@your_site.your_net

If you feel you need to reach a human, send email to

undertow-approval@majordomo.law.emory.edu

POSTING

To post a note to the list, send email to: undertow@law.emory.edu
