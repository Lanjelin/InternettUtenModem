Hints for
     Installation

As a browser you can use either the Microsoft Internet or Netscape version 2.0 onwards. The Internet explorer will recognise automatically if you need Plug-in and if necessary, will install it immediately itself. If you use Netscape then please load the newest version of Plug-in which is compatible with your system:

PC/Win95 (146 KB) - for 32-Bit Netscape-Version
PC/Win3.11 (138 KB) - for 16-Bit Netscape-Version
Mac68 (49 KB sit-packed) - for normal Macs
MacPM (79 KB sit-packed) - for Power-Macs

To start up the loaded data double click. The installation programme will then ask you where it should install the Plug-in. In your Netscape folder there is a folder called "Pug-ins," in the same file as "Netscape.exe," it must be in there. If you have more Netscape versions installed please be sure to choose the newest one. Then start up Netscape afresh - that's it!
