Mailing lists are frequently used through the role-playing community in order to convey information concerning a particular subject, game world, etc. To subscribe to them, simply click on the hypertext link and type in the command following that in the subject header.

For a more complete list of mailing lists there are many list of lists available which keep track of all the new and old mailing lists along with the ones which have disappeared. IRPS also holds one of these lists of lists in it's directories.

    Al-Qadim Mailing List: type "subscribe qadim-l [youremail]"
    Darksun Mailing List: type "subscribe darksun [youremail]"
    Forgotten Realms Mailing List: type "subscribe realms [youremail]"
