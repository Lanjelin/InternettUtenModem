Getting PGP 2.6.3i by email

If you, for some reason, are not able to download a copy of PGP via WWW or FTP, then it is possible to have it sent to you via email. The files are encoded using base64 (MS-DOS), HexBin (Mac) and UUencode (Unix).

Hypnotech PGP file server

The easiest way to get a copy of PGP by email, is probably to send a message to pgp@hypnotech.com (Subject: HELP for instructions). You can do so manually, or fill in the form below:
Please send
to my email address at:
NB! Some of the files are very big (more than 800 Kb). Not all mail systems can handle messages of this size. If you don't get a reply, this is probably the reason.

FTP by email

There also exists a number of email servers that can retrieve FTP files for you automatically. These usually have the possibility to split the uuencoded file in several parts. Here's an example:
  1. Send a message to ftpmail@ftpmail.ramona.vix.com
  2. In the body of the message (no subject), write: 
     reply <yourmail@domain.com>
     connect ftp.ifi.uio.no
     binary
     uuencode
     get /pub/pgp/pc/msdos/pgp263i.zip
     quit
