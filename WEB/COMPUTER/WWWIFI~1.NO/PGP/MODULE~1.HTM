Language modules for PGP

This page contains links to all foreign language modules for PGP 2.6.3 and 2.6.3i that are available at the present time. Please note that there are modules for other languages around as well, but these are generally outdated and not adapted to PGP 2.6.3(i).

PGP has already been translated into the following languages:

Brazilian Portuguese
Chinese (GB & BIG5)
Czech
Danish
Dutch
English (UK)
Finnish
French
German
Greek
Hungarian
Italian
Japanese
Lithuanian
Malay
Norwegian
Polish
Romanian
Russian
Serbian
Spanish
Swedish
These language modules contain all the necessary translations for both PGP 2.6.3 and 2.6.3i. If you have a language module for an earlier PGP version, an upgrade is strongly recommended. The files are also available from:

ftp.encomix.es
ftp.ox.ac.uk
ftp.elte.hu

In order to get PGP to use your national language, you must:

  1. Download the desired language module from the list above
  2. Unpack the files into your PGP directory
  3. Update the language line in your CONFIG.TXT so that it reads:

    language = xx

    where xx is the 2-letter code for your language

If you would like to translate PGP to a language not listed here, please read these guidelines first.
