PGP tools & add-ons

The following is a list of some of the many tools and utility programs that exist to make life easier for the PGP user. See also: PGP compatible libraries. If you know of another program that you think should be on this list, please let me know.

MS-DOS

PGPShell 3.3 - menu-driven interface for PGP.EXE
PGP Menu 3.0 - menu-driven interface for PGP.EXE
Pretty Good Privacy Shell 1.99a - menu-driven interface for PGP.EXE
PGP-Front 0.99 - interactive shell for PGP.EXE
PGP Shell & Message Editor - use PGP without remembering all the command line switches
MPI 1.0 - small shell for PGP, similar to EZ-PGP
PGP for Idiots 3.0 - menu-driven interface for PGP.EXE
Offline AutoPGP - integrates PGP encryption into your QWK or SOUP reader
BlueKey 2.0 - a PGP shell for offline mail readers, such as BlueWave, GoldEd and Termail
PGPWave - shell for use with Blue Wave and other offline readers
MPGP 1.01b - uses PGP for conventional encryption of multiple files
PGPSort - sorts your keyring and removes bad keys

Windows

There are so many PGP utilities for Windows around, that I have moved them to a separate page.

OS/2

PGP Easy Rider - front-end shell for PGP
pmPGP - a PM Shell for PGP
Gibbon PGP Front-End - for the EPM editor
PGP REX - Rexx scripts that add easy PGP usage to OS/2 mailers
MR/2 ICE & PGP - Rexx scripts to integrate PGP with the MR/2 ICE mail reader
PMMail - PGP aware email client

Unix

exmh - Tcl/Tk-based front-end to mh which is MIME-compliant and PGP-aware
Bryce's Auto-PGP - script to integrate PGP encryption in mail readers (elm, pine, mh, trn etc.)
papp (Pine And PgP) - Perl script for using PGP with Pine
Integrating Pine and PGP - simple but useful csh script
MailCrypt - an Emacs/PGP Interface
Emacs auto-pgp - an Emacs/PGP Interface
Script which integrates pgp in zmail
elmpgp - PGP interface for elm reader
PrivTool - PGP-aware replacement for Sun mailtool (SunOS/Linux)
Joes Mailtool - Add-on for Sun mailtool (SunOS)
PGPsendmail - PGP-aware replacement for Unix sendmail
PGPwho - computes the signature distance to another person

VMS

Using PGP From TPU on VMS Systems

Amiga

PGPtoGUI - GUI for PGP

Macintosh

Envelope 1.0- Provides single click encryption and decryption of your Eudora email messages
MacPGP Control - AppleScript application that offers an easy-to-use, more Macintosh friendly user interface to MacPGP
MacPGP Accessories - from MCIP
Encryplet - AppleScript droplet which makes it easy to encrypt and decrypt files by drag-and-dropping them onto a Desktop Encryptor icon
SafeMail - PGP compatible mailer

Source code (platform independent)

PGP Tools - a PGP crypto construction set
Stealth - strips headers from PGP messages
MandelSteg and GIFExtract - hide PGP data in fractal GIF images
