PGP Language support

PGP 2.6.3i is a very international piece of software, and both the program and the documentation have been translated into several languages.

Documentation

The PGP User's Guide has been translated into several languages.

Brazilian Portuguese [Text | MS Word 2.0 | MS Word 6.0]- translated by Roberta Paiva Bortolotti
German [HTML]- translated by Christopher Creutzig
Italian [HTML | Text]- translated by Marco Giaiotto
Spanish (volume 1 only) [Text] - translated by Armando Ramos
If you intend to translate the PGP User's Guide into a language not listed here, please contact me.

Language modules

PGP can easily be set up to use other languages than English when displaying internal messages, errors etc. All you need is a language module consisting of a LANGUAGE.TXT file and accompanying help file for your particular language. Language modules for PGP 2.6.3(i) are currently available for more than a dozen different languages. Unfortunately, the language modules don't work with the Macintosh version of PGP.
