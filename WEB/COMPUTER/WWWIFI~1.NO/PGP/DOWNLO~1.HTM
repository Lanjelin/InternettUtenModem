Downloading PGP

The latest international version of PGP is version 2.6.3i. This version is only intended for users outside the US. If you live in the USA, you should use PGP 2.6.3 or PGP 2.6.2 from MIT instead.

Note: PGP 2.6.3i is currently at patchlevel 'a'. Most of the source code and binary distributions below have been updated with the recent changes.

Platform availability

PGP 2.6.3i is available as precompiled binaries for the most popular platforms, including MS-DOS, OS/2, Amiga, Atari, Macintosh, BeOS and Linux (Slackware/RedHat). If you're using Windows, please look here before downloading.

You may also download the source code and compile PGP yourself. You may want to do this if you:

  1. Are on a Unix platform
  2. Are really paranoid, and want to check the source code for any backdoors, etc.
  3. Are just curious, and want to learn more about cryptography
  4. Want to port PGP to a new platform

Downloading
