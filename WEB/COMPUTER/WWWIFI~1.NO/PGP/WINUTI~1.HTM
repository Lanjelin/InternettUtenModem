PGP tools & add-ons for Windows

There are dozens of PGP utility programs for Windows around, but as I don't use PGP under Windows myself, I cannot say which is best. Try out for yourself, or ask someone else for advice. :-) Corrections and additions to this list may be sent to stale@hypnotech.com.

If you haven't already installed PGP:

  1. Download PGP for DOS (the 16-bit version will run on almost any computer)
  2. Open a DOS box under Windows, install PGP, read the documentation, set environment variables and reboot your computer
  3. Download (shift + click on the file link) and install your choice of Windows shell

Front-end shells

These programs offer a user-friendly interface to PGP. Most common PGP functions are carried out using point-and-click rather than cryptic command line parameters. Some of the tools listed here may also be used with email software, esp. those that support clipboard operation (cut & paste).

The client-based, transparent extension has been fully integrated into the familiar email user environment. It automatically recognizes encrypted emails and offers an easy-to-use keyring administration. CryptoEx works with both the US and the international Version of PGP. A user-friendly adminstrator tool helps distributing CryptoEx and managing user options in large Windows networks.

Email integration

These utilities have been specifically designed to integrate PGP into your email environment, without using cut & paste. Some of them lack key management functions.

  • License: Commercial, try before you buy
  • Home page: http://www.compulink.co.uk/~net-services/pgp/
  • Platforms: Windows 3.x/NT/95
  • Description: Direct interface to your e-mail software - you don't have to cut and paste. Extensive key management including Key Server functions. Mollusc currently supports Eudora Pro (v2.2 - 32 bit), Eudora Light (1.5.2 and 1.5.4), Pegasus Mail 2.40, WinCIM 2.0, Ameol, Free Agent and Netscape 1.2N.

PGP compatible tools

Tools that offer PGP compatibility without actually using PGP.

INAS (It's *Not* A Shell)

  • Current version: 1.0.2
  • Author: Jonas Hagmar
  • File: inas102a.exe (821 KB)
  • License: Freeware
  • Home page: http://home2.swipnet.se/~w-21488/
  • Platforms: Windows NT/95
  • Description: INAS is a Windows 95 program that is able to read and write PGP-compatible files. It provides nearly all the functionality of PGP, including encryption, decryption, signing and key management, but does not rely on PGP or the PGP code for any of its processing. The program is coded entirely in Delphi Pascal and inline assembler. It's currently availiable only in a freeware version intended for non-commercial use only.
