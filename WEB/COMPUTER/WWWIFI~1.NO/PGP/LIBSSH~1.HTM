PGP compatible programs and libraries

The following is a list of programs and libraries that implement PGP functions without having to call the PGP program itself. You can use these programs to create your own PGP-compatible programs.

PGPTools: PGP library for C
Cryptix-Perl: PGP library for Perl
Cryptix-Java: Cryptographic extensions for Java
pgplib: Experimental DLL which implements common PGP functions
INAS (It's Not A Shell): PGP-compatible Windows 95 application, written in Delphi Pascal
