What's New with Spanky

It can get confusing browsing these fractal pages in their semi-organized state brought on by my random updating. All the links can be a little overwhelming so I'm going to make an attempt to filter all the new links through this page before moving them to their appropriate index page. If you're just looking for recent developments in the Fractal Community, then this is the page for you. I'll form a small set of links under each of the main information catagories I maintain and just keep pointers to recent additions here. I'll try and keep the number of links low and reasonably up-to-date. This page will have a very simple format, within each catagory, new things will be added to the bottom of the list, and moved to the related index page when they get to the top.

This is also one page where the header image may change from time to time. I'll use this page to introduce any current favourite or recent image that I've created.

Interactive Fractal Servers

Other Interactive Servers directory

  • Prof. Alberto Strumia strumia@ciram3.cineca.it has a Webpage dedicated to Virtual Maps of Mandelbrot and Julia Sets the images are precalculated but it is a very informative roadmap to these fractal types.
  • John Olsson provides an interactive website that generates a fractal worldmap in GIF-format and also provides links to retrieve his sourcecode.
  • Neal Kettler has created a new version of his interactive Fractal Explorer and it is much quicker and much more versatile than in the past.
  • Simon Arthur chroma@mindspring.com has implemented a Mandelbrot viewer written in Java. If you have a Java capable browser, try it out.
  • Ken Shirriff shirriff@eng.sun.com also provides java-script to Zoom in on the Mandelbrot . If you can run Java applets, its an interesting interface. Source is also provided.
  • Nick Lilavois nlilavois@cc.atinc.com introduces, Java Man. Not the anthropological homonid, but a web-based mandelbrot generator using the Java language. If your browser supports this extension, take a look!
  • A nice and simple Java Applet to display the Sierpinski Triangle
  • Mark Mcclure mmcclure@wlu.edu from the Department of Mathematics at Washington and Lee University in Lexington Virginia has two interesting fractal pages. He has created a Java based Interactive Julia Set generator and also a a random animation of julia sets. (non-interactive).
  • Gary Kerbaugh MACCCOLL@ECUVM.CIS.ECU.EDU has written a really inovative interactive fractal server that draws IFS type fractals. It has a fixed set of traditional IFS designs but it also has a user interface to let you create your own IFS fractals. He calls his server Fractals: Not by the Numbers and it is a well designed and documented interactive fractal server.
  • Zak Elliott zelliott@iterated.com at Iterated Systems informed me of a new IFS Java applet developed there.
  • James Henstridge james@daa.com.au in Australia has created a great series of Java Applets for Fractals with about a dozen different types. His source code is also available.
  • Michael G. Henderson mgh@nis.lanl.gov at Los Alamos has a well established Interactive Mandelbrot/Julia zoomer available.
