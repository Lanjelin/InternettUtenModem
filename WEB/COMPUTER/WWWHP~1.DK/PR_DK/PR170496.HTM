Hewlett-Packard integrerer

UNIX- og Windows NT miljøer

Ny produktlancering fra Hewlett-Packard giver kunderne mulighed for at opbygge integrerede UNIX- og Windows NT løsninger. Med HP's Avanced Server/9000 kan brugerne benytte en HP UNIX maskine som NT fil- og printserver.

HP kan nu levere AT&T's Avanced Server og NT's NOS server (Network Operation System) på sine UNIX-baserede servere og arbejdsstationer. Foruden NT's sikkerhedsrutiner på brugerniveau, adgangskontrol og netværkskontrol, understøtter serveren følgende NT-klienter: DOS, Windows NT, Windows 95, Windows til Worksgroup og LAN Manager.

Brugere, der vælger at opbygge et blandet UNIX- og NT miljø, kan afvikle begge serverfunktioner på én og samme maskine. Brugerne sparer hermed investeringen i en separat NT-server, får exceptionelle udvidelsesmuligheder af serverkapaciteten og den driftsikkerhed og pålidelighed, der kendetegner UNIX verdenen. Med Advanced Server/9000 tilbyder HP det bedste fra to verdener på en UNIX maskine - fuld support af NT baserede klienter og afvikling af UNIX applikationer.

- Annonceringen er endnu et bevis på HP's satsning på både UNIX og Windows NT som serverløsninger. For HP drejer det sig ganske enkelt om at kunne give vores kunder den bedste løsning hver gang - uanset om den er baseret på UNIX, Windows NT eller et blandet miljø, siger produktchef Niels Thomassen, Hewlett Packard Danmark.

- At HP seriøst investerer i begge forretningsområder viser det fælles udviklingsprojekt mellem Intel og HP om en ny mikroprocessor. En fælles mikroprocessor for Intel og HP vil give HP en særdeles stærk position på såvel UNIX som NT markedet, siger Niels Thomassen.

Advanced Server/9000 er baseret på AT&T's Advanced Server for UNIX Systems (AS/U), version 3.51.
