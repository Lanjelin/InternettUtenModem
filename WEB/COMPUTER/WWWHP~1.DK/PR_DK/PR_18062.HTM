Hewlett-Packard annoncerer nye Unix-servere og nyt

revolutionerende disk array system med AutoRAID

Hewlett-Packard udvider sit program af Unix-servere i den lave ende af server markedet. De nye Unix-servere, HP 9000 D260 og D360, har en ydelse, der ligger 60% højere end de tidligere HP-servere i samme klasse. Samtidig annonceres HP's nye revolutionerende disk array baseret på AutoRAID teknologien. De nye disk array kan tilkobles alle HP 9000-servere og arbejdsstationer.

HP's disk array med AutoRAID er sidste skud på stammen af HP's løsninger til datalagring. AutoRAID teknologien er udviklet af HP og indeholder mere end 25 nye patenter. Fordelen ved de nye disk array er bl.a., at de er fejltolerante og selvkonfigurerende til et optimalt RAID niveau.

AutoRAID konceptet er specielt velegnet til systemer, hvor performance og sikkerhed er kritiske faktorer. AutoRAID kan med fordel benyttes til "mission-critical" OLTP systemer, systemer til elektronisk handel, kundesupport, on-line information og produktions- kontrol.

De nye Unix-servere og AutoRAID teknologien er nærmere beskrevet i vedlagte samt på http://www.hp.com.
