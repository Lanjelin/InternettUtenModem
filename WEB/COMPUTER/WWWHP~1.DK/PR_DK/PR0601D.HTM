150 millioner "hits" om dagen

på én Internetserver

I en netop offentliggjort performancetest har Hewlett-Packard's Internet server, HP Domain Server model K460, opnået 1.827 SPECweb96 operationer per sekund. Det svarer til, at serveren kan betjene 150 millioner opslag om dagen fra brugere på Inter- eller intranettet. HP's K460 er den første server nogensinde, der når mere end 1.000 operationer per sekund i en standard SPECweb96 test.

HP's Domain server er specielt konfigureret som Internet- og intranetserver. Domain Server model K460 er baseret på en HP 9000 maskine med 4 stk. 64-bit HP PA-8000 CPU'er, Internetsoftware, værktøjer til system- og netværksadministration samt software til sikker dataudveksling på Inter- og intranettet.

SPECweb96 er en standard client/server baseret WEB test, som viser det maksimale antal forbindelser en WEB server kan supportere med acceptable svartider. Testmetoden blev annonceret i juli 1996 af Standard Performance Evaluation Corporation (SPEC). SPEC er desuden kendt for andre standard tests af computersystemer, eksempelvis SPECint95 og SPECfp95.

Resultatet for HP Domain Server model K460 er 87% bedre end den nærmeste konkurrent, Digital's Model 4100, og 300% mere end SUN's bedste resultat, opnået med en SUN Netra I3.1 model 2200.

Med SPECweb96 resultatet for K460, har HP SPECweb96 rekorderne for både RISC/Unix og Microsoft Windows NT miljøer.

Se detaljer om SPECweb96 tests på SPEC's WEB: http://www.specbench.org
