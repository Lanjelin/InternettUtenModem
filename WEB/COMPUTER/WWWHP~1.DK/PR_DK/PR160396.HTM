Hewlett-Packard eneste leverandør med

praktisk viden om ny SAP R/3 teknologi

Hewlett-Packard står særdeles stærkt på SAP markedet efter en succesfuld og omfattende test af SAP's nye ALE (Application Link Enabling) teknologi. HP kan nu tilbyde ekspertise og rådgivning om implementering af SAP løsninger i blandede EDB-miljøer, hvor både UNIX og Windows NT indgår.

ALE, der blev lanceret sammen med SAP R/3 release 3.0, binder SAP systemer sammen på tværs af maskintyper, databaser, R/3 releases, netværk og 3. parts software. En virksomhed kan med ALE etablere ét fælles SAP miljø baseret på fysisk adskilte SAP systemer, der kører i hver deres separate EDB miljø.

HP's test blev udført i tæt samarbejde med SAP i HP/SAP kompetencecentret i Walldorf i Tyskland. En HP9000-K UNIX maskine med SAP og Oracle fungerede som hovedkontor, mens 2 HP LS netservere med Windows NT, SAP og Oracle fungere som salgs- og produktionsafdelinger. Systemerne var forbundet via 64K- og 19.2K Bit kommunikationsforbindelser.

Målingerne omfattede bl.a. skalérbarhed, ydeevne, ressourceforbrug på de enkelte delsystemer, automatisk genstart efter netværksnedbrud og transaktionsbelastninger. Hertil kommer de generelle erfaringer med opsætning og optimering af SAP i blandede SAP miljøer baseret på HP udstyr.

- De nye testresultater stiller HP i en unik position. Vi har nu, som den eneste leverandør, den know how, der skal til for at implementere fysisk adskilte SAP systemer i én integreret løsning, siger produktchef Allan Rixen, Hewlett-Packard Danmark.

- Det giver kunderne frihed til at vælge den optimale implementering af SAP i forskellige dele af en virksomhed samtidig med at virksomhedens helhedsinteresser tilgodeses. En afdeling på Sjælland vælger måske Windows NT og Oracle som grundlag for deres SAP løsning, mens hovedkontoret i Jylland vælger SAP under UNIX og Informix. HP's SAP-konsulenter tilbyder den ekspertise og rådgivning, der binder de forskellige afdelinger, kontorer, fabrikker, etc. sammen i ét samlet virksomhedssystem, siger Allan Rixen.
