Hewlett-Packard og Oracle tilbyder

Web-baseret Data Warehouse løsning

Brugerne får nu fuld adgang til Data Warehouse systemer via deres Web-browser. Løsningen er baseret på HP's OpenWarehouse Web koncept, Oracle's Universal Server og diverse Oracle-applikationer og værktøjer.

For at give kunderne lettere og mere fleksibel adgang til Data Warehouse løsninger, annoncerede HP i april måned OpenWarehouse Web. Med OpenWarehouse Web kan brugerne, fra deres normale Web-browser, administrere og arbejde med centrale og decentrale Data Warehouse løsninger. Hermed kombineres Internettes fleksibilitet og styrke med Data Warehouse-løsningernes mange forretningsmæssige muligheder.

Løsningen inkluderer en Web-version af HP's administrationsprogram til Intelligent Warehouse og udvidet sikkerhedsrutiner med HP's "Web security" teknologi. Samtidig tilbyder HP, via partnerløsninger, Web-baseret data-adgang, data-analyse, data-mining og applikationsudvikling. Følgende Oracle Web-løsninger er nu fuldt integreret i HP's Data Warehouse Web koncept:

* Oracle Universal Server under HP Unix til håndtering af alle typer data.

* Oracle Web Server og Oracle PowerBrowser til udvikling af Internet- og

Intranet applikationer.

* Web-baseret version af Oracle Express Server til support af Intranet-baserede

OLAP applikationer.

* Oracle Parallel Server til "mission-critical" Web-applikationer.

* Oracle udviklingsværktøjer som "Designer/2000 Web Server Generator".

Hewlett-Packard tilbyder en komplet Data Warehouse løsning, Intelligent Warehouse. HP's Intelligent Warehouse består af en række konkrete Data Warehouse produkter samt af HP's konsulentafdeling, der er specialiseret i implementering af Data Warehouse løsninger. Via samarbejdspartnere, komplementerer HP sin Data Warehouse løsning inden for nøgleområder som database, data-analyse, data-mining og ledelsesinformation.
