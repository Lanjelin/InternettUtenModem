Unison Maestro

Produktbeskrivelse

Unison Maestro er det førende værktøj på markedet til styring af jobafvikling i et heterogent IT-miljø. Maestro kan afvikle jobs overalt på netværket i henhold til standard tidsskemaer eller tidsskemaer tilpasset det specifikke miljø. Udover tidsfaktoren kan opgaver afvikles afhængigt af andre faktorer og betingelser, og således tilpasses komplekse driftmiljøer. Maestro kan afvikle opgaver på tværs af platform og understøtter bl.a. UNIX, Windows NT, MVS og OS/2. Endvidere tilbydes tæt integration til specielle applikationer f.eks. SAP/R3. Maestro er fuldt integreret med HP OpenView platformen og giver derved simpel administration fra ét centralt sted.

Styrker

  • High availability ved hjælp af fejl-tolerante agenter
  • Skalérbar, håndterer afvikling af op til 20.000 jobs
  • Udnytter ressourcer effektivt på tværs af platform
  • Høj sikkerhed, automatisk start af recoverjobs ved eventuelle fejl
  • Integreret med HP Openview IT/Operations
  • Integreret med SAP/R3
  • Understøtter HP-UX og Windows NT
