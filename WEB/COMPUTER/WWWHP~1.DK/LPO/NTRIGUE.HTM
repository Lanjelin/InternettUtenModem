NTrigue

Produktbeskrivelse

NTrigue er løsningen for kunder, som ønsker at integrere et Unix/NT miljø på en økonomisk effektiv måde. NTrigue giver brugeren mulighed for at afvikle hhv. Windows NT, Windows 95 og Windows 3.1 applikationer på UNIX arbejdsstationer, X-terminaler eller PC'ere med X-terminal software. NTrigue er baseret på Windows NT, og afvikles på en central Intel-baseret NT-server (f.eks. en HP Netserver). Brugerens applikationer afvikles således på denne server, mens grafikken håndteres via desktop-klientens X-interface. Dette koncept giver maksimal performance og kompatibilitet, og tillader således brugeren at afvikle 16- og 32-bits Windows applikationer uanset hardware-platform og uden yderligere investering i desktop hardware eller software.

Styrker

  • Windows applikationer på alle desktop-platforme uanset hardware
  • Applikationer afvikles i native-kode, ingen emulering
  • Performance er uafhængig af desktop-platformen
  • Mulighed for integration af Windows-applikationer i WEB-sider via NTrigue
  • Central styring og overvågning af applikationer og server
