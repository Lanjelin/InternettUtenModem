SoftWindows

Produktbeskrivelse

SoftWindows for UNIX er et softwareprodukt, der gør UNIX arbejdsstationer i stand til at fungere som en fuldt ud kompatibel PC. Dette betyder, at alle Microsoft Windows og MS-DOS applikationer kan køre på arbejdsstationen, side om side med UNIX-applikationer, og at der kan kopieres og indsættes tekst og grafik mellem de forskellige applikationer.

Styrker

  • Microsoft Windows 3.11 og MS-DOS 6.22 er pre-installeret og pre-konfigureret i SoftWindows for UNIX, så installationen kan foretages på få minutter.
  • SoftWindows for UNIX har fuld PC netværksfunktionalitet til brug med enten standard Ethernet eller Token Ring hardware, og PC e-mail systemet kan derfor anvendes samtidig med at der køres UNIX-applikationer.
  • SoftWindows for Windows anvender flydende licenser, og licensomkostningerne kan derfor minimeres.
