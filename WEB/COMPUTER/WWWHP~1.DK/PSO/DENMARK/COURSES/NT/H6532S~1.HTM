• Supporting Microsoft® Windows 95 TM

    H6532S

    Formål:

    Målet med dette kursus er, at give deltagerne de nødvendige færdigheder til at supportere Windows 95.

    Deltagere:

    Personer med ansvar for administration og/eller drift af Windows 95 miljø , herunder installation, konfigurering, vedligeholdelse, og problemløsning.

    Forkundskaber:

    Kendskab til MSDOS, Windows 3.xx, samt til almene begreber såsom CPU, BIOS, og netværk.

    Varighed:

    5 dage.

    Indhold:

    Overskrift

    • Installation og konfigurering.
    • Multitasking, processer, prioriteter, virtuel hukommelse og systemarkitektur.
    • Gennemgang af princippet "Plug and Play".
    • Netværk herunder Microsoft Networks, Novell NetWare, TCP/IP, Microsoft Exchange (MAPI).
    • Printere og netværksprintere.
    • Programmer og arkitekture, og indbyrdes relationer, tuning af et MSDOS-baseret program.
    • Skærm-driver model, konfigurering og fejlfinding. Implementering af multimedia.
    • Diskopsætninger og filsynkronisering.
    • Dial-up, klienter, protokoller, fejlfinding.

    Pris:

    14,250.
