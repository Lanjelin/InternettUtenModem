• Supporting Microsoft ® Windows NT TM 4.0 Core Technologies

    H6582S

    Formål:

    Målet med kurset er, at sætte deltagerne i stand til at installere, konfigurere, optimere, integrere og fejlfinde alle centrale dele i Windows NT 4.0.

    Deltagere:

    Personer med ansvar for implementering og support af Windows NT Workstation version 4.0 og NT Server version 4.0.

    Forkundskaber:

    Kendskab til Windows 95 eller Windows NT 4.0 brugergrænsefladen samt viden svarende til kurset H6581S (Administering Windows NT 4.0) .

    Varighed:

    5 dage.

    Indhold:

    Overskrift

    • Installation og konfigurering af Windows NT.
    • Brugeradministration.
    • Filsystemer og Windows NT 4.0 miløet
    • Disk partitioner og tilhørende administration.
    • Fejlsøgning af applikationer herunder Win 16, Win32 og MSDOS.
    • Konfigurering af netværksprotokoller.
    • Gennemgang af netværks-services herunder browser, DNS, WINS of DHCP.
    • Remote Access Services (RAS og TAPI).
    • Internet og intranet samt NetWare integration.
    • Printerstyring fra Windows NT.
    • Konfigurering af klienter og netværksmiljø .
    • Windows NT fejlsøgning.
