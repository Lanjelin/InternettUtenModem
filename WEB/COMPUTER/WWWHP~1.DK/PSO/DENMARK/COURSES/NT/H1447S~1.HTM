• Supporting Microsoft® Windows NT TM 4.0 Enterprise Technologies

    H1447S

    Formål:

    Målet med dette kursus er, at give deltagerne de nødvendige færdigheder til at kunne designe, implementere, feljfinde og supportere et større Windows NT 4.0 netværk med flere domains.

    Deltagere:

    Personer med ansvar for administration og drift af større installationer baseret på Windows NT 4.0.

    Forkundskaber:

    Kendskabsvarende til kurser H6582S "Supporting Windows NT 4.0 Core Technologies".

    Varighed:

    5 dage.

    Indhold:

    Overskrift

    • Windows NT 4.0 miljøet
    • Windows NT Server 4.0 Directory Services.
    • Én-vejs og to-vejs Trust relationer .
    • Adm. af brugere og grupper på tværs af Trust's. .
    • Effektive placeringer af servere.
    • Implementering af effektiv synkronisering.
    • Betragtninger i forbindelse med implementering af Directory Services.
    • Brug af Perfomance Monitor'en.
    • Oprettelse af overvågnings-database.
    • Performance og System bottlenecks analyser.
    • Ressource allokering og analye i Windows NT.
