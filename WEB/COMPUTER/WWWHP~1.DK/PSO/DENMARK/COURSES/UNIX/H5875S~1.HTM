• HP-UX 10.x system- og netværksadministration for erfarne UNIX administratorer

    H5875S

    Formål:

    Dette kursus er beregnet for erfarne UNIX administratorer, der flytter til et HP-UX-miljø, eller som skal have HP9000 servere/arbejdsstationer integreret i et eksisterende netværk.

    Teori / øvelser: 60 / 40

    Deltagere:

    Erfarne UNIX systemadministratorer, der ikke har erfaring med HP-UX.

    Forkundskaber:

    Deltageren skal have 1 - 2 års erfaring som UNIX systemadministrator.

    Varighed:

    5 dage.

    Indhold:

    HP9000 systemhardware

    • PA-RISC arkitektur
    • Hardwarekomponenter
    • Hardwarekonfiguration

    Initialisering, systemstart og nedlukning

    • Bootprocessen
    • Gennemgang af systeminitialiseringsprocessen
    • Beskrivelse af processen for brugerlogin
    • Nedlukning af HP-UX
    • Brug af HP VUE brugerinterface

    Device filer

    • Brug af device filer
    • Block og character devices
    • Brug af mksf, insf, lssf og rmsf
    • Automatisk oprettelse af device filer

    Brugeradministration

    • Oprette, fjerne og rette brugere i systemet
    • Ændre brugernes passwords
    • Forældelsesproces for passwords
    • Standardopsætning for brugere

    Filsystemet

    • Gennemgang af superblock og Inoder
    • Typer af UNIX-filer
    • Forskellige leverandørers filsystemstrukturer
    • Kommandoer til administration af filsystemer
    • Brug af fsck og fsdb til problemløsning

    Administration af diskareal

    • Diskpartitionering
    • Brug af logical volumes
    • Oprettelse af filsystemer
    • Mount og unmount filsystemer

    Logical Volume Manager (LVM)

    • Gennemgang af LVM konceptet
    • Oprettelse af volume groups, logical volumes og physical volumes
    • Brug af disk striping og LVM

    Administration af filsystemer

    • Opsætning og brug af access control lists
    • Brug af symbolske links
    • Administration af dump filer og systeml og filer

    IP Connectivity

    • ARPA services
    • Oprettelse af hostnavn på et HP-UX system
    • Konfiguration af et netværks interface på et HP-UX system
    • Konfiguration af routing på LAN
    • Værktøjer til problemløsning

    Network File System (NFS) og Automounter

    • Beskrivelse af funktioner i NFS
    • Installation af NFS
    • Mount og unmount NFS filsystemer
    • Konfigurering og brug af automount

    Network Information Service (NIS)

    • Gennemgang af NIS-funktioner
    • Konfiguration af NIS-server og -klient
    • Konfiguration og brug af NIS i et hetorogent UNIX-miljø

    Netværkssikkerhed

    • Opsætning af sikkerhed i forbindelse med brug af netværkskommandoer
    • Gennemgang af specifikke filer til brug for netværkssikkerhed

    Domain Name Service (DNS) og BIND

    • Muligheder og funktioner med DNS
    • Opsætning af BIND servere og klienter
    • Forskellige leverandørers metoder til opslag af hostnavn ved brug af DNS og NIS

    Sikkerhedskopiering

    • Backupmuligheder med standard- kommandoer tilgængelige i HP-UX
    • Komprimering af filer og data
    • Flytning af filer mellem UNIX systemer i et netværk

    Printeradministration

    • Kommandoer til printeradministration
    • Brug og vedligeholdelse af spooler systmerne BSD, AIX eller System V

    Administration af systemressourcer

    • Monitorering af systemressourcer
    • Begrænsning i brug af systemressourcer

    Kerneadministration

    • Grundlæggende forskelle i administration af kernen mellem BSD og System V
    • Konfiguration af en ny kerne
    • Problemløsning i forbindelse med kernekonfiguration

    Installation af HP-UX

    • Forskel mellem installation og opdatering
    • Gennemgang af installationsprocessen for HP-UX 10.x

    Installation af software

    • Forberedelse til installation af software
    • Brug af swinstall
    • Kontrol af softwareinstallation
    • Netværksinstallation af software
