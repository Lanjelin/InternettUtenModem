HP OpenView Network Node Manager for netværksansvarlige

    B4743S

    Formål:

    At sætte deltageren i stand til at overvåge og administrere større netværkssystemer ved hjælp af HP OpenView Network Node Manager (OpenView NNM). Desuden sættes deltageren i stand til at installere, konfigurere og fejlsøge OpenView NNM.

    Teori / øvelser: 50/50

    Deltagere:

    Netværksadministratorer, som er ansvarlige for overvågning og administration af større netværk.

    Forkundskaber:

    Deltageren skal have deltaget på HP-UX 10.x systemadministration (H6296S) og HP-UX netværksadministration (H6295S) eller have tilsvarende viden.

    Varighed:

    5 dage.
