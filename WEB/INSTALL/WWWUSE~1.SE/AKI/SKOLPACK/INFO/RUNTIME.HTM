Information om runtime filerna

Varför behövs det så kallade runtime filer för att köra SkolPack och varför ingår de inte i huvud installations filen?
Runtime filerna behövs för att SkolPack har utvecklats med hjälp av ett högnivåprogrammeringsverktyg, Visual Basic. Runtime filerna kan antingen skickas med inbakade i installationen eller separat. I SkolPack's fall har jag valt att skicka dem separat. Det innebär att du måste ladda ner två filer om du inte redan har runtime filerna för VB 4.0 32-bit installerade i din dator. Varför skickar jag då inte med runtime filerna för att det ska bli enklare? Jo, dels slipper du ladda ner filerna om du redan har dem och om det senare kommer en mindre uppgradering av SkolPack så behöver du inte ladda ner runtime filerna en gång till utan då räcker det med att bara ladda ner SkolPack. Eftersom det är två filer att ladda ner blir det också mindre att ladda ner igen om du skulle ha oturen att förbindelsen bryts.
Ett tips till dig som är osäker på om du har runtime filerna installerade i din dator är att ladda ner SkolPack först och installera det och prova om du kan köra programmen. Kan du inte det så måste du även ladda ner runtime filerna och installera dem också.
