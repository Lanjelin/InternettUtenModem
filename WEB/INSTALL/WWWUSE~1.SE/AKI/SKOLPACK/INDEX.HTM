   SkolPack: Fem skolprogram i ett paket
-För dig som studerar eller bara vill lära dig någonting!

Allt detta ingår

Fem svenska sharewareprogram för Windows 95 och NT 4.0!

Glosan - Ett glosprogram för att träna glosor.
OrdBoken - Skapa ordböcker, sök efter ord
FrågeSport - Skapa en frågesport och spela den.
Plump - Förbättra din huvudräkning.
Hänga Gubbe - Gissa ord och undvik att bli hängd.

Ladda ner SkolPack redan idag!

A. Den senaste versionen (april: nyheter) i en fil:

Klicka här för att ladda ner SkolPack (2.8 MB).

B. Förra versionen i två filer:

1. Innan du kan köra programmen som ingår i SkolPack behöver du några så kallade runtime filer.
Klicka här för att ladda ner runtime filerna (1.2 MB) .
När du sedan ska installera SkolPack kör du ovanstående fil först och den installerar runtime-filerna åt dig.

2. Fortsätt att ladda ner SkolPack.
Klicka här för att ladda ner SkolPack (1.6 MB).
På samma sätt som i ovanstående fall kör du filen för att installera SkolPack.

Extra saker för dig som redan använder SkolPack

A. En stor engelsk<>svensk ordbok till OrdBoken
Klicka här för att ladda ner: BigEnSv.exe (378 kB)
Kör filen och välj katalogen där du installerade SkolPack och klicka på Unzip. Starta därefter OrdBoken och välj den nya ordboken.

B. Tysk<>engelsk ordbok till OrdBoken
Klicka här för att ladda ner: DeEng.exe (47 kB)
Kör filen och välj katalogen där du installerade SkolPack och klicka på Unzip. Starta därefter OrdBoken och välj den nya ordboken.
