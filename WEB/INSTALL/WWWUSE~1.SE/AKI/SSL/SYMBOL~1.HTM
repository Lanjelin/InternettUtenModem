Vad betyder symbolerna?

Programmets namn - Ingen symbol, men ändå viktigast. Det är på programmets namn du ska klicka på när du vill ladda ner programmet.

- Programmet går att köra med Dos

- Programmet går att köra med Windows 3.1 och högre

- Programmet går med Windows 95 och högre.

- Programmets version följer efter denna symbol.

- Programmet distribueras i en självupppackande exe-fil, efter att du har laddad ner den kör du bara filen, enkelt.

- Programmet distribueras i en Zip-fil. Måste packas upp innan användning med tex WinZip.

- Klicka på denna symbol för att gå till programmets hemsida.

- Klicka på denna symbol för att skicka e-mail till programmets tillverkare.

- Programmet kräver ytterligare filer för att köras, tex runtime filer för VB. Klicka på symbolen för att ladda ner filerna.
