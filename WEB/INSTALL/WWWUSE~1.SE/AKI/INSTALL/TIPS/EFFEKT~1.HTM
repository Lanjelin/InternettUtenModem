Tips

Effektiva tips

För att du ska kunna använda Windows effektivt bör du känna till nedanstående tips. Det är inte tips där du behöver ställa in något, utan tips som du har nytta av direkt.

Använd hjälpen


Det första ställe som du bör leta efter hjälp är direkthjälpen som finns i nästan alla Windowsprogram. Det enklaste sättet att komma åt hjälpen är att trycka på tangenten F1 på det ställe där du vill ha hjälp, i nya finessrika program får du då hjälp direkt om det du vill, annars får du välja det du vill läsa om från innehållet.

Hjälpen kan också nås från Hjälp knappar och menyn Hjälp eller ? i programmet.

Växla program


Håll ned och tryck

Glöm inte bort att du kan köra flera program samtidigt i Windows. Från och med Windows 95 är det enklaste sättet att växla program att använda växlingsfältet, som normalt visas längs nere på skärmen. Ett annat sätt att växla program, som även finns i Windows 3.1x är att använda Alt och Tab tangenterna. För att växla program håller du nere Alt (på vänster sida) och trycker sedan på Tab (över CapsLock), då ser du en text på mitten av skärmen, fortsätt att trycka på Tab tills du kommer till rätt program, då släpper du Alt-tangenten.

För att växla program, kan du också minimera de program som du inte använder för tillfället, men det fungerar inte då du har uppe en dialogruta då måste du använda ett annat sätt att växla program.

En dialogruta är ett fönster som saknar max- och minimeringsknappar. Det går inte heller att klicka på programmets fönster och få det att hoppa fram, utan då måste använda OK eller Avbyt-knapparna i en dialogruta för att få bort den.

Styr Windows med tangentbordet


Håll ned och tryck

För att komma åt en meny eller något annat element med en understrykning under en bokstav ska du hålla nere Alt och trycka på bokstaven med understrykningen. Det blir samma effekt som om du hade klickat på elementet med musen.


För att komma åt knappar och andra element ska du använda Tab en eller flera gånger, när du trycker på Tab ser du hur markeringen runt elementet flyttar på sig, det element som är markerar kan sedan ta emot information från tangentbordet.


Shift och Piltangenterna

När det gäller att markera nånting med tangentbordet, gör du det med piltangenterna. Text markeras genom att hålla nere samtidigt som du med piltangenterna väljer det du vill markera.


Enter Mellanslag Esc

Med Enter eller mellanslag "klickar" du på någonting med tangentbordet. Esc är ofta det samma som avbryt.

Öppna filer på olika sätt

Filer kan öppnas på många olika sätt.

  • Arkiv menyn i programmet, då måste du först starta programmet där du ska öppna filen
  • Dubbelklicka på en fil, då startar programmet och öppnar filen automatiskt.
  • Dra och släpp. Dra en fil och slätt den på programmets fönster.

Använd en metod som passar bäst för tillfället, men se upp för alla program stöder inte alla dessa metoder.
