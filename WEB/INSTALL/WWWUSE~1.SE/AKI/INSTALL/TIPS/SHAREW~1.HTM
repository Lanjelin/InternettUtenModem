Shareware program

Efter att du har trimmat din Windows med tips eller om du vill förbättra den med hjälp av något program eller om du bara vill ha ett litet program för att visa bilder och text är det dags att titta på sharewareprogrammen.

Sharewareprogram är program som man får kopieras helt fritt, men om du vill använda programmet efter att du har testat det ska du betala en summa pengar till programmeraren. Det kallas att registrera programmet, ibland får du en också förbättrad version av programmet när du registrerar dig.

Några exempel

Det finns massor med sharewareprogram för alla tänkbara ändamål, allt från små verktygsprogram som fyller luckorna från stora vanliga program till kompletta ordbehandlare och ritprogram.

Ett av dem första sharewareprogram som du bör skaffa dig är någon sorts program som packar upp filerna i vilka sharewareprogrammen distribueras, det gäller särskilt om du hämtar program från Internet eller BBS:er. WinZip är just ett sådant program. Det kan packa upp Zip filer som ar det vanligaste formatet på packade filer. WinZip kan också packa upp andra filer (t.ex. arj-filer), men då krävs det att du har Dos versionen av uppackningsprogrammet vilkens fil du ska packa upp.

Ett sharewareprogram som faktiskt står i klass med de vanliga programmen är Paint Shop Pro. Det är ett grafikprogram med vilket man kan öppna och spara många olika grafikformat, ändra färger i bilder, redigera och manipulera bilder.

Powertoys är en samling program som ger Windows 95 extra finesser. T.ex. medföljer det ett program som heter QuickRes. Med det kan man enkelt och snabbt byta upplösnings på skärmen utan att starta om Windows. I Powertoys ingår också ett program som utökar autoplay funktionen för CD-Rom skivor, ett program för att ställa in "dolda" finesser i Windows 95. T.ex. kan man ställa in hur snabbt startmenyn ska dyka upp på skärmen. Det ingår också en drivrutin med vilket man kan skriva ut html sidor från vilket program som helst, samt några andra mindre program.
