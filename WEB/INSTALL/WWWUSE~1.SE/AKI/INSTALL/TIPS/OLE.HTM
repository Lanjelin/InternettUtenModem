OLE i Windows

OLE betyder Object Linking and Embedding, Objekt Länkning och Inbäddning. Det innebär att du kan ha en bild i ett dokument och sedan dubbelklicka på bilden ör att redigera den med ritprogrammet. OLE finns i två varianter. I den första varianten startas programmet i ett eget fönster när du dubbelklickar på ett objekt. I den andra varianten som kallas OLE2 ändras programmets menyer, verktygsfält mm när du dubbelklickar på ett objekt och det kan då redigeras på plats. OLE kräver mycket minne och du bör därför försöka undvika använda det på datorer med lite minne.

Änsålänga är OLE ganska långsamt och klumpigt, men i framtiden är det tänkt att denna teknik ska användas så att man arbetar med dokumenten i centrum istället för programmen som idag. Redan i Windows 95 arbetar man med dokument, men det finns ju en startmeny för program så det går även att arbeta med programmen som vanligt.
