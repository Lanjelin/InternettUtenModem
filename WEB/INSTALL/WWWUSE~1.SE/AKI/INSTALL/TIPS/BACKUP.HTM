Säkerhetskopiering

Äntligen är du klar med all installation och alla inställningar. Nu är det dags att se till att du inte får göra om allt som du gjort hittills. För att förhindra att dina dokument, program och inställningar går förlorade av någon orsak bör du säkerhetskopiera ditt system regelbundet.

Till största delen är det du själv som kan orsaka olika fel och då är det enkelt att bara återställa från säkerhetskopian, men även om hårddisken skulle gå sönder kan du återställa allt om du har en bra säkerhetskopia.

Olika möjligheter att säkerhetskopiera

Till disketter

Passar bra för att säkerhetskopiera dina egna dokument och inställningar, men om du ska kopiera alla program också kommer det gå åt många disketter och mycket tid. Du måste sitta och byta diskett vid kopieringen. Disketter är fortfarande det vanligaste sättet att lagra data, vilket ger dig fördelen att du kan ta med din säkerhetskopia var som helst och återställa den där.

Till band

För att säkerhetskopiera till band behöver du en bandstation, som kostar ungefär 1500 kr och uppåt beroende på kapacitet. Att säkerhetskopiera till band är ett billigt och ganska snabbt alternativ, eftersom du kan låta kopieringen pågå medan du gör något annat.

Till annan hårddisk

Det förmodligen enklaste sättet att säkerhetskopiera är att köpa en likadan hårddisk till som man sedan kopplar till ett löstagbar hållare. Då kan ta bort hårddisken och förvara den någon annanstans då du inte kopierar till den. Säkerhetskopieringen går helt enkelt till genom att du kopierar allt som finns på din vanliga hårddisk till säkerhetshårddisken. Om nu oturen skulle vara framme och din vanliga hårddisk skulle gå sönder kan du bara ersätta den med din extra hårddisk.
