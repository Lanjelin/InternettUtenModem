Inställningsfilerna i Windows

Känner du till kontrollpanelen i Windows? Från det kan du ändra många inställningar i Windows. Var lagras alla inställningar som du gör i program?

Inställningsfilerna i Windows är Registret, Win.ini, System.ini och ini-filer för program.

Registret

Från och med Windows 95 är det tänkt att alla inställningar ska lagras i registret. Genom att köra programmen regedit.exe kan du enkelt titta och ändra på inställningarna, innan du börjar ändra i registret bör du ta en säkerhetskopia av det. Den kan du sedan återställa om det visar sig att systemet inte startar igen efter att du har gjort ändringar i registret.

Registret finns även i Windows 3.1x, men används i mindre omfattning, främst till OLE inställningar. För att se hela innehållet i Windows 3.1x måste du köra regedit med parametern /v.

I Windows 95 finns även de andra inställningsfilerna kvar för att alla gamla program ska kunna köras, men de används inte av själva Windows 95, utan för Windows 95 inställningar är det registret som gäller.

Om Ini-filer

Till skillnad från registret är ini-filerna vanliga textfiler och kan ändras med alla editorer som kan spara filer i textformat. Dessutom kan Win.ini och System.ini ändras med hjälp av programmet sysedit, som också medföljer Windows.

En ini-fil är uppbyggd av sektioner. En sektion kan se ut så här:

[Position] 
X=15 
Y=0

Överst visas rubriken inom hakparanteser, under rubriken kommer sedan ett antal variabler med tilldelade värden. I ovanstående sektion kan du ändra programmets X eller Y position genom att ändra värdena 15 eller 0.

Win.ini

I denna ini-fil lagras inställningar som gäller allmänt i Windows 3.1x, men öven andra program kan lägga till sektioner med egna inställningar. Antingen rör det sig om inställningar om var programmet finns eller andra viktiga inställningar.

System.ini

Här finns inställningar för olika drivrutiner och andra hårdvaroberoende inställningar.

Ini-filer för program

Dem flesta program skapar en ini-fil med samma namn som programmen förutom ändelsen som är .ini istället för .exe. Dessa ini-filer för program lagras i Windowskatalogen. Både inställningar som du gör i dialogrutor i programmen och andra dolda inställningar kan lagras i programmets ini-fil.

I Kontrollpanelen ändrar du dem vanligaste inställningarna utan att behöva tänka på ini-filer.
