Att tänka på vid installation av program

Många program installerar inte bara filer till sin egen katalog utan också till Windows och Windows-Systemkatalog och gör däröver ändringar i diverse systemfiler. För att du senare ska kunna avinstallera ett program helt måste du veta vilka filer det installerade till dessa kataloger.

Det finns program som hjälper dig med detta. WinClean är ett program som vid installationen kollar vilka filer som installeras. Det kan sedan senare ta bort filerna. Ett annat program som man inte behöver köra då man installerar program är CleanSweep, som kan analysera vilka filer ett program använder och sedan kan användaren välja vilka som ska tas bort. Om du ofta testar många program så kan det vara en bra idé att använda ett avinstallationsprogram för att slippa en massa onödiga filer i din dator.
