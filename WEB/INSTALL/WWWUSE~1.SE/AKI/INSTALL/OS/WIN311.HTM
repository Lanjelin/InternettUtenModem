Att Installera och ställa in Windows 3.11

Här följer först en fortsättning till Dos-genomgången. Nu ska Windows 3.11 installeras så att du får maximalt med hårddiskutrymme och bästa prestanda.

Det går också att installera Windows 3.1 på samma sätt, men i den versionen finns ingår inte alla program som nämns.

  1. Vid installationen väljer du anpassad installation för att kunna välja bort delar som du inte behöver använda.
  2. Nu får du välja hur du vill installera Windows. I rutan som kommer fram ska du avmarkera installera program som finns på hårdisken. Avmarkera också installera skrivare om du inte har någon skrivare.
  3. Nu får du välja vilka filer du vill installera. För att spara plats kan du till exempel välja bort:
    • Mail i Windows 3.11 tar stor plats och bör väljas bort om det inte behöver användas.
    • Microsoft at Work fax likaså.
    • Schedule+ är också ett stort program.
    • Självstudier kan du ta bort om du redan kan Windows.
    • Terminalen kan du ta bort om du inte har ett modem eller om du använder ett annat terminalprogram.
    • Välj fritt vilka hjälpfiler, spel och skärmsläckare som du vill installera.
  4. Fortsätt nu med installationen tills du kommer till en ruta där installations-programmet frågar efter ändringar i AutoExec.bat och Config.sys. Du ska nu spara ändringarna till dem och fortsätta sedan tills installationen är klar.
  5. När installationen är klar ändrar du sökvägarna till filerna i Windows-katalogen i Config.sys och AutoExec.bat till filer i Dos-katalogen. När det t.ex. står C:\Windows\Himem.sys så ändrar du det till C:\Dos\Himem.sys. Alla filers sökvägar utom ifshlp.sys (som inte finns i Dos-katalogen) ska du ändra till Dos istället för Windows, då kan du nämligen ta bort följande filer från Windowskatalogen:
  • emm386.exe, msd.exe, smartdrv.exe, himem.sys, ramdrive.sys och *.txt. För att detta tips ska fungera bra måste du ha Windows 3.11 och Dos 6.2 eller 6.22.
  1. Nu kan du starta om datorn och sedan starta Windows. För att du ska få bästa prestanda ska du ställa in virtual minnet som permanent och sätta på både 32-bitars fil och disk-åtkomst. Du gör det från kontrollpanelens 386-avancerat. Cacheminnets storlek kan du också minska om du har lite minne i din dator. Nu kan du också ta bort Smartdrive från din AutoExec.bat eftersom den inte längre behövs då du använder Windows egen disk-cache.

Windows 3.11 är igång.
