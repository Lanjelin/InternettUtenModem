Att Rensa och ställa in Dos

Denna genomgång av vad som man kan ta bort och ställa in i Dos gäller först och främst MS-Dos 6.22, men eftersom även Windows 95 bygger på Dos så lämpar sig även en del tips för det.

Denna genomgång är också bra för att få så mycket ledigt plats som möjligt på hårddisken i äldre datorer som har mindre hårddiskar och som inte heller kan köra nya operativsystem som Windows 95.

Börja först med att göra en komplett installation av Dos om du inte redan har gjort det.
