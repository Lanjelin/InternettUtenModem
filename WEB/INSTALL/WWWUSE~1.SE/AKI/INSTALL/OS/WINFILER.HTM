Filerna som Windows byggs upp av

För att bättre förstå vad som händer när du installerar program behöver du veta hur Windows är uppbyggt. Det hjälper också dig när du ska leta efter fel.

De olika komponenterna i Windows är:

  • Win.com
  • Kärnfilerna: Kernel, User och GDI
  • Typsnitt och drivrutiner
  • Filer för Dos-stöd
  • Tillbehörsprogram

Win.com

Kollar datortyp och minne, laddar och sätter sedan ihop kärnfilerna, typsnitten, drivrutinerna, språkfilerna, dosstödet och andra filer beroende på datorn, och startar sedan Windows.

Kärnfilerna

Kernel: Hanterar minnet och maskin-resurserna. Kör och kontrollerar program.

User: Hanterar gränssnittet, skapar, flyttar och förstör fönster och andra delar av gränsnittet.

GDI: (Graphic User Interface) Ritar grafiken på skärmen.

Typsnitten

Typsnitt är filer som ger texten utseende. En fil som hör till ett typsnitt har någon av ändelserna .ttf, .fon eller .fot.

Systemtypsnitt är typsnitt med ändelsen .fon och används normalt i menyer, dialogfönster och för visning av text i andra delar av Windows. Storleken, antalet punkter i dessa typsnitt är förutbestämd.

TrueType-typsnitt är skalbara typsnitt och till varje typsnitt krävs två filer: en .ttf fil och en .fot fil. Dessa typsnitt kan användas både till visning av text på skärmen och för utskrifter. P.g.a. skalbarheten uppkommer inte heller några taggar eller trappor vid utskrifter, vilket är ett vanligt fenomen vid utskrifter av icke skalbara typsnitt.

Drivrutiner

Varför behövs det drivrutiner? För att förklara det tar jag ett exempel med skrivare. Olika skrivare har olika metoder för att skriva ut bokstäver, men för att det inte ska behövas alla dessa metoder att skriva ut i alla program, har Microsoft i Windows skapat ett gränssnitt för utskrifter, när ett program t.ex. ska skriva ut bokstaven A anropar det utskriftsrutinen i Windows, vilken sedan översätter anropet så att skrivaren förstår det och kan skriva ut bokstaven. Detta gör att man kan skapa program och sedan skriva ut till vilken skrivare som helst bara det finns en drivrutin till den. Att det finns en drivrutin till en enhet är alltså lika viktigt som att det finns en anslutningssladd.

Windows behöver drivrutiner för allt. Vanliga drivrutiner till olika enheter som skrivare, grafik- och ljudkort osv kommer du säkert att behöva installera eller byta ut någon gång. Utöver det använder Windows drivrutiner för portar, tangentbord och liknande maskin-nära enheter. Dessa drivrutinerna medföljer Windows och installeras helt automatisk, men dem kan bytas ut vid behov. Dem flesta drivrutiner har ändelsen .drv, men dem kan även ha andra ändelser t.ex. .386, .vxd och dessutom använda sig av dll filer.

Ett bra tips är att använda dem senaste versioner av drivrutiner. Nya drivrutiner kan du t.ex. hämta från Internet.

Tillbehörsprogram

För att du själv ska kunna göra något med operativsystemet behövs tillbehörsprogram. Tillbehörsprogram som medföljer Windows är t.ex. dem vanliga programmen Program- och Filhanteraren och de andra verktygen i Windows.
