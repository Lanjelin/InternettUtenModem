Operativsystemet

Innan du kan använda din dator till någonting måste du installera ett operativsystem. Det är nästan alltid förinstallerat när du köper en dator. Är du en van användare kan det vara bra att installera det själv för att få med allt som behövs.

De vanligaste operativsystemen

Här följer en genomgång av dem vanligaste operativsystemen för PC.


Dos
Det första operativsystemet för PC, är kommandobaserat och används till teckenbaserade program och spel. Kör ett program i taget. Finns i flera varianter från olika tillverkare.


Windows
Det berömda grafiska fönstergränssnittet för PC, kommer från Microsoft och finns för närvarande i tre olika "stora versioner".

3.11 - Körs på Dos och kan köra 16-bitars program. Används framförallt på gamla datorer som inte klarar av dem nyare operativsystemen. Kan köra flera program samtidigt (multitasking) även om det inte alltid går bra.

95 - En fristående version med inbyggd Dos, kan köra gamla 16-bitars program med bra kompatibilitet, men för att du ska kunna utnyttja alla nya finesser måste du ha nya 32-bitars program, vilka är speciellt utvecklade för 95:an och NT. Lite högre systemkrav än 3.11, men har bättre multitasking, snyggare utseende och långa filnamn. Ett bra alternativ för hemmaanvändare.

NT - Utvecklat för hög säkerhet, både när det gäller intrång och körning av 32-bitars program. Har också bra multitasking och kan köra de flesta gamla Windows-program, och vissa DOS-program, men med mindre bra kompatibilitet p.g.a. säkerheten. Spel för DOS är inte att tänka på. Finns också i en serverversion för nätverk. NT:S säkerhet gör det till ett bra alternativ för företag och programutvecklare. De stora nackdelarna är dock systemkraven och den dåliga kompatibilteten med DOS-program.


OS/2
Ett operativsystem från IBM som kan köra både gamla Dos och 16-bitars Windowsprogram, men också program som har skrivits speciellt för OS/2. OS/2 programmen kan antingen vara teckenbaserade eller grafiska. OS/2 har bra multitasking och har därför blivit populärt hos sysopar som med OS/2 kan ha både BBS:en igång och samtidigt använda datorn till någonting annat. Ungefär samma systemkrav som för Windows 95.

Jag förutsätter i fortsättningen att du har Windows (3.11 eller 95) som operativsystem i din dator.
