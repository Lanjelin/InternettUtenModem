Vad är The Words?

The Words är ett lättanvänt program för att träna ord till exempel dina engelska glosor.
Du skriver in dina egna ord med programmet och sparar dem. Sedan kan du förhöra dig orden ungefär som i ett vanligt förhör.
För mer information titta på bilder av programmet.

Ladda ner The Words

The Words är freeware och finns i två versioner: 16- and 32-bit. Den senaste versionen är 1.1. Nytt är att du kan välja dina egna typsnitt i programmet.

The Words 16-bit version [218 kB] (Zip-fil Klicka på denna länk för att ladda ner)
För Windows 3.1 och Windows 3.11, du måste också ha runtime filen VBRUN300.DLL installerad innan du installerar The Words (16-bit). Klicka här för att ladda ner en zip-fil som innehåller runtime filen [223 kB].
The Words 32-bit version [452 kB] (Zip-fil Klicka på denna länk för att ladda ner)
För Windows 95 och NT 3.51 och högre, du måste också ha VB 4.0 32-bit runtime filerna installerade innan du installerar The Words (32-bit). Klicka här för att ladda ner en exe-fil som installerar runtime filerna när du kör den [1.2 MB]. Ett annat ställe där du kan ladda ner runtime filerna.
