De eventuella ekonomiska förluster som ett OS skulle kunna orsaka kan inte bli så stora att de på något sätt skulle kunna rubba stabiliteten i Sveriges ekonomi. ett OS i Sverige kommer rycka många försoffade ungdomar in i idrotten som med idrotten som bas kan bli Sundare människor så väl fysiskt som socialt bara detta borde snart ha sparat tillbaka de pengar som os eventuellt kan gå back. Vi måste se långsiktigt på detta och inte bara se på vad siffrorna säger en månad efter OS är över, De säger ingenting om vad det betytt för Sverige.
