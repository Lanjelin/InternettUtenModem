About Our Site Design

At the present time we have chosen to stay as close to the HTML 2.0 standard as possible so that we can reach the largest possible audience. We have recently begun using the 'table' extension. If our tables appear garbled, consider upgrading your browser. The 'table' tag greatly enhances menuing without eating bandwidth. We highly recommend that you use the Netscape Browser (at least version 2.0); the latest version can be downloaded directly from Netscape.
