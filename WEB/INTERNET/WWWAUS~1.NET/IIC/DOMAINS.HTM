OVERVIEW
In the Introduction to the Internet chapter you have learned how an internet domain address looks like. The list below provides you with over a 100 country and other domains, which will help you to determine, which country a computer with a particular address is located. If the host is not located in the USA, then the type of organization or business running this server is often indicated by adding either .co for commercial, or for non profit organization and ac for academic before the country domain. So an academic institution in Austria would have

ac.at

as the domain. A non profit organization in the UK would have

or.uk

as the domain and a commercial business in Italy would have

co.it

as the domain name. Please note that this are only suggested conventions and that there might be some servers, which are not following these conventions, but most of them do. By using this list you can also find out, from which country an e-mail has been sent.
