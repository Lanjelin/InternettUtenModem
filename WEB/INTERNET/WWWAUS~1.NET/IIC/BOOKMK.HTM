ZIP File, 3KB

After downloading the file, extract it to any directory of your choice. Be careful not to overwrite your original bookmark file. You can use this bookmark file by simply selecting Open from your Web Browser. The bookmark will then be displayed in your browser window. If you are using Netscape Navigator, select Bookmark, Edit Bookmarks and then Open. Note that your existing bookmarks will be not available any more from that time. However, they are not deleted, you just need to open your original bookmark file in the way described before. If you want to mix this bookmark with your existing bookmarks, then you follow the same procedure as described above. But do not select Open; choose Import instead. Note that this might lead to name conflicts which might result in some bookmark entries to be overwritten.

NOTE: We do not take any responsibility if you loose any of your exisintg bookmarks due to one of the procedures described above. We recommend viewing the bookmark as a normal HTML File.
