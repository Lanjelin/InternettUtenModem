ActiveX

BEDEUTUNG:

Hierbei handelt es sich um eine neue Technologie, die von Microsoft entwickelt wurde. Es handelt sich dabei um den Nachfolger von OLE. Mit dieser Technologie ist es möglichm OLE Elemente in eine Web Seite einzubauen, die nicht in HTML enthalten sind. Diese Elemente werden automatisch heruntergeladen und installiert, wenn ein Besucher eine Seite mit einem solchen Element besucht. Teil von ActiveX ist auch VBScript, eine Scriptsprache für das World Wide Web.
