WWW

BEDEUTUNG:

wird oft abgekürzt als: WWW, W3
Ein Internetdienst mit einer graphischen Benutzeroberfläche. Der Benutzer kann von einer Web-Seite zur nächsten surfen indem man auf Hyperlinks klickt. Die einzelnen Seiten müßen nicht auf dem selben Server liegen.
