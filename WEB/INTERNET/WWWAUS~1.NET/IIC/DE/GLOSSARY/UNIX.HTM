Unix

BEDEUTUNG:

Ein Betriebssystem, welches sehr oft auf Servern verwendet wird. Es erlaubt das anlegen von verschiedenen Benutzern mit unterschiedlichen Zugriffsrechten. Jeder Benutzer dem es erlaubt ist, kann sich auf den Server einloggen und dort Resourcen verwenden. Um einen Unix Rechner verwenden zu können müssen Sie sich einen Usernamen und ein Passwort besorgen. Sie benötigen außerdem einen Telnet Client um über das Internet auf diesen Rechner zugreifen zu können.
