FTP

ABKÜRZUNG:

File Transfer Protocol.

BEDEUTUNG:

Mit diesem Dienst können Sie Dateien von einem Computer auf einen anderen übertragen. Viele Software Archive für fast alle Betriebssysteme sind über FTP erreichbar. Um von einem entfernten Computer Daten übertragnen zu können, müssen Sie einen Username und ein Passwort kennen. Viele Software Archive lassen sich über das sogenannte Anonyme FTP ansprechen.
