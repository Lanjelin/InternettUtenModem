Eine Web Site installieren
23. Dezember. 1996

1. WAS KANN ICH TUN ?

Sie möchten also Ihre Produkte, Ideen oder sich selbst am Internet präsentieren und wissen nicht so ganz wie Sie das anstellen sollen ? Dieser Artikel beschreibt die elementaren Schritte, Überlegungen und Anforderungen die Sie vornehmen müssen, um dieses Ziel zu erreichen.

2. WAS BENÖTIGE ICH DAFÜR ?

Sie haben eine Idee - und nun benötigen Sie jemanden, der HTML Seiten erstellen kann und Sie bringen sich HTML selbst bei. Hier sind einige gute Links, wenn Sie sich HTML beibringen möchten:

Microsoft Site Builder Workshop
The World Wide Web Consortium

Dann benötigen sie einen Computer, der permanent mit dem Internet verbunden ist. Auf diesem Computer legen Sie Ihre Dateien ab. Es gibt verschiedenste Wege, einen solchen Computer zu "erhalten", die im folgenden diskutiert werden:

EIN WEB SERVER
Sie benötigen also einen Web Server, auf dem Sie Ihre Dateien ablegen können. Es gibt verschiedenste Möglichkeiten Festplattenplatz auf einem solchen Computer zu erhalten, je nach dem wieviel Geld Sie investieren möchten. Wenn Sie eine nicht komerzielle Homepage erstellen möchten, dann können Sie einen der Gratis Homepage Dienste in Anspruch nehmen. Solche Dienste finden Sie unter den folgenden Adressen:

Geocities

Allerdings bieten diese gratis Homepage Dienste oft nicht die Möglichkeit, CGI Scripts wie Zugriffsstatistiken und E-Mail Formulare zu verwenden.

Die zweite Möglichkeit wäre, Festplattenplatz von einem ISP zu mieten. Dies mag für viele komerzielle Zwecke der beste Weg sein. Die Kosten, die dabei anfallen sind von Provider zu Provider verschieden und hängen meistens davon ab, welche zusätzlichen Optionen Sie in Anspruch nehmen möchten. Generell werden die Kosten entweder aufgrund des von Ihnen benötigten Festplattenplatzes oder des Datentransfers, der durch Ihre Site entsteht, berechnet. Wir empfehlen die erste Art der Abrechnung. Bevor Sie jedoch Ihre endgültige Entscheidung für einen ISP treffen, sollten Sie folgende Fragen stellen:

Wie kann ich die Seiten aktualisieren ? Mit E-Mail oder FTP ?
Bieten Sie bereits fertige CGI Scripts, wie Mail oder Zugriffszähler ?
Kann ich meine eigenen CGI Scripts verwenden ? Welche Progrmmiersprachen unterstützen Sie?
Bieten Sie detailierte Zugriffsstatistiken ?
Kann ich meinen eigenen Domain beantragen ?

Beachten Sie bitte, daß zusätzliche Dienste, wie zum Beispiel CGI Scrpts zusätzliche verrrechnet werden. Es gibt mittlerweile viele Provider, welche die obigen Dienste anbieten. Wenn Sie allerdings keinen Provider in Ihrer Nähe finden können, dann sollten Sie ruhig einen Provider in einem anderen Land versuchen.

Die dritte und möglicherweise teuerste Methode ist das Server Housing. Dabei wird Ihr eigener Computer bei einem ISP untergestellt und permanent mit dem Internet verbunden. Sie müssen allerdings den Computer selbst installieren und konfigurieren. Der Vorteil diese Option ist, daß Sie die volle Kontrolle über Ihren Computer besitzen. Bevor Sie diese Option in Erwähgung ziehen, sollten Sie allerdings die Kosten kalkulieren. Sie benötigen:

... einen eigenen Computer mit viel RAM und schneller CPU
... ein eigenes Betriebssystem
... Ihre eigene Server Software
... eine eigene IP Adresse und Domain Namen
... eine gemietete Verbindung mit dem Internet

Wenn Sie die volle Kontrolle über Ihren Computer und CGI Scripts haben möchten und wenn sie planen Geld mit Ihrer Web site zu machen und viele Besucher erwarten, dann ist diese Option wahrscheinlich das beste, ansonsten zahlt sich diese Investition nur selten aus.

3. DIENSTE

Im folgenden finden Sie eine detailierte Beschreibung von Dienste, die Ihr ISP anbieten sollte.

AKTUALISIEREN IHRER SEITEN
Im allgemeinen gibt es zwei Möglichkeiten Ihre Seiten zu aktualisieren. Die beste Methode ist sicherlich FTP. Wenn Sie FTP verwenden, dann können Sie Ihre Seiten updaten, wenn Sie es möchten. Das bedeutet auch, daß Ihre Besucher die Änderungen sofort sehen, sobald Sie eingespielt worden sind. Sie können außerdem schneller auf bestimmte Umstände reagieren; zum Beispiel das ausbessern von Fehlern. Wir empfehlen nicht das aktualisieren via E-Mail. Dazu müssen Sie Ihre Seiten zuerst packen und zu Ihrem Provider schicken. Dann wird Ihr ISP die aktualisierten Dateien installieren. Zwischen der Absendung und der tatsächlichen Aktualisierung vergehn meistens ein bis zwei Tage, bevor die Seiten aktualisiert werden und Sie können die Seiten wahrscheinlich nicht an nicht Werktagen aktualisieren.

ZUGRIFFSSTATISTIKEN
Zugriffsstatistiken sind wichtig, da es ansonsten keinen anderen Weg gibt, mit dem Sie feststellen können, wieviele Besucher Ihre Seite hat. Detailierte Statistiken sollten auf wöchentlicher oder monatlicher Basis verfügbar sein und Ihnen exakte Auskunft geben, wie oft jede Seite Ihre Site besucht wurde. Für das IIC bekommen wir jede Woche einen detailierten Bericht, aus dem wir ersehen können, wie oft eine bestimmte Seite anefordert wurde. Dazu gibt es tägliche, stündliche und monatliche Statistiken. Außerdem können wird feststellen, welche Seiten aus welchen Ländern wie oft angefordert wurden. Wenn Ihr ISP keine Statistiken anbietet, dann können Sie einen der folgenden kostenlosen Dienste benutzen:

Digits
Internet Audit Bureau

FEHLER LOGS
Niemand ist perfekt. Ein Errorlog hilft Ihnen dabei, Fehler in Ihren Dokumenten zu finden (speziell ungültige Links). Ein solches Errorlog sollte Auskunft geben, welcher Fehler in welchem Dokument aufgetreten ist.

CGI SCRIPTS
Sie möchten sicherlich Feedback von den Besuchern Ihrer Site erhalten. Um ein solches Feedbackformular zur Verrfügung zu stellen, benötigen Sie ein CGI Script. Viele Provider beiten fertige CGI Script für diesen Zweck an. Des weiteren gibt es Möglicherweise fertige Scripts für einen Zugriffszähler und ein Gästebuch. Es ist ganz nützlich ein solches Feedbackformular zur Verfügung zu stellen. Wenn Sie jedoch Ihre eigenen Scripts für diesen Zweck verwenden möchten, fragen Sie Ihren ISP, welche Programmiersprachen zu Erstellung solcher Scripts er unterstützt. Perl ist zum Beispiel eine oft verwendete Programmiersprache (auf Unix und Windows NT Servern) für diesen Zweck.

IHR EIGENER DOMAIN
Wenn Sie Ihre Web Site erstellt haben, dann möchten Sie vielleicht Ihren eigenen Domain Namen statt dem Namen Ihres ISPs und einem Unterverzeichnis. Es gibt zwei Arten, wie Ihr Domain aussehen kann. Zum Beispiel www.IhrDomain.com/Verzeichnis oder nur www.IhrDomain.com. Um die zweite Art zu verwenden, benötigen Sie einen virtuellen Server und kostet vielleicht etwas mehr als die erste Option. Um Ihren eigenen Domain zu erhalten, kontaktieren Sie Ihren ISP oder eine der folgenden Organisationen:

Internic International, USA
Asia Network Information Center
Danish Network Information Center
NIC Pakistan
NIC France
NIC Mexico
Liechtenstein and Switzerland
German NIC

4. FÜR IHRE SITE WERBEN

Sobald Sie Ihre Site online haben, müssen Sie potentielle Besucher auf Ihre Site aufmerksam machen. Nur indem Sie Ihre Site online bringen, wird sich niemand diese Site ansehen, da niemand weiß, daß Sie existiert. Melden Sie Ihre Web Site bei unter anderem bei den folgenden Suchmaschinen an:

Yahoo
Lycos
Hotbot
Magellan
Webcrawler
Alta Vista
Opentext
Promote It

Zusätzlich dazu schicken Sie die Adresse Ihrer Web site and verschiedene andere Verzeichnisse, wie zum Beispiel iGuide. Des weiteren können Sie führ Ihre Site werben, indem Sie Werbung auf anderen, bereits sehr bekannten, Web Sites schalten. Dazu müssen Sie eine Grafik erstellen, die dann von der Web Site direkt auf Ihre Site verweist. Die Kosten dafür belaufen sich je nach Site zwischen 50 USD und 1000 USD oder mehr. Wenn Sie Ihre Site auf anderen Sites bewerben möchten, dann suchen Sie eine Site, die bereits oft besucht wird und kontaktieren Sie den Autor dieser Site für detailierte Informationen. Das IIC bietet Ihnen zum Beispiel die Möglichkeit günstig Werbung zu schalten. Achten Sie außerdem darauf, daß die Site, auf der Sie werben möchten einen guten Ruf hat; Sie möchten Ihre Werbung doch nicht auf einer Sex Site, oder ?
