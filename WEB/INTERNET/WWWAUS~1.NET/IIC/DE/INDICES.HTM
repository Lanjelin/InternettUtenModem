Site Indices
12. Jänner 1997

Hier sind Links zu verschiedenen Site Indices und Verzeichnissen. Dabei handelt es sich um gute Startpunkte für Ihre Erforschung des Internets.

Frauenspezifische Seiten im WyberNetz
Adresse: http://www.web-publishing.de/women/seiten.htm
Ein Verzeichnis von speziellen Frauenseiten in Englisch, Deutsch und Italienisch.

Blue Planet
Adresse: http://www.erols.com/owl/planet/homepage.html
Links zu hochwertigen Sites am Web

1111 Links
Adresse: http://www.ziff.com/~pcpro/1111/1111_idx.html
Diese Links wurden vom deutschen PC-Professional zusammengestellt.

Brians Revised Internet Yellow Pages
Adresse: http://www.mindport.net/~b_ruth/
Eine ansehnliche Liste von vielen Internet Sites. Auf der Startseite finden Sie verschiedene Kategorien. Ein Klick auf eine Kategorie bringt Sie auf eine weitere Site mit vielen Links zu der entsprechenden Kategorie.

Frank's World
Adresse: http://www.geocities.com/SiliconValley/5043
Hierbei handelt es sich um die ersten HTML Versuche eines Studenten. Neben persönlichen Infos finden Sie viele Interessante Links hier.

Hackstadts Hierachical Hotlist
Adresse: http://www.cs.uoregon.edu/~hacks/hotlist/
Diese Site bietet eine Liste von Links, die normalerweise von anderen Verzeichnissen nicht angeboten werden. Viele Kategorien und Sub-Kategorien.

Hidden Rivers of the World Wide Web
Adresse: http://www.netaxs.com/~hopkins/
1500 Links zu excellenten Sites auf dem Web. Organisiert mittels Kategorien.

The Hot Hot List
Adresse: www.ccsf.caltech.edu/~roy/others.html Eine einfache Seite, die nur aus Text besteht - keine Bilder. Daher sehr schnell.

Hot Links
Adresse: http://fly.hiwaay.net/~fbeaver/
Viele Links in 8 Kategorien.

Hot Links! Reference
Adresse: http://fly.HiWAAY.net/~fbeaver/reference.html
Viele Links - wiederum in Kategorien gruppiert.

Hotsheet
Adresse: http://www.tstimpreso.com/hotsheet/
Dutzende Links zu interessanten Web-Sites. Enthält keine Grafiken - lädt sehr schnell.

Internet Tourbus
Adresse: http://www.worldvillage.com/tourbus.html
Homepage einer großen und interessanten Mailinglist. Graphisch sehr gut designed. Sie können hier den Internet Tourbus abonnieren (Mailingliste).

Linkmaster
Adresse: http://linkmaster.com/
Eine interessante Site mit vielen Features wie Java, Frames und Real Audio.

A2Z Lycos
Adresse: http://a2z.lycos.com/
Ein in Kategorien unterteiltes Verzeichnis einer bekannten Suchmaschine. Millionen Links !

Megalist WWW List
Adresse: http://www.worcester.com/cgi-bin/megalist/megalist
Eine weitere Site, die Frames verwendet. Keine großertigen Graphiken; daher sehr schnell. Verschiedenste Kategorien: Art, Business, Computers, Education, Economy, Government, Health, Internet, Java, Legal, news, Politics und Reference.

Net Links
Adresse: http://www.interlog.com/~csteele/netlinks.html
Interessante Links. Einige Links zu Sites speziell für Einsteiger (höhrt sich vertraut an, oder ? ).

Netscape Internet Directory
Adresse: http://www.netscape.com/escapes/internet_directory.html
Eine Verzeichniss der Entwickler des Netscape Navigator. Nicht sehr viele Links, aber dafür sehr gute.

Netsurfer Digest Home Page
Adresse: http://www.netsurf.com:80/nsd/
Homepage des Netsurfers Digest. Eine wöchtentliche Mailingliste mit vielen interessanten Sites. Ein Archiv bisheriger Aussendungen wird ebenfalls geboten.

New Rider's Official World wide Web Pages
Adresse: http://www.mcp.com/newriders/wwwyp/index.html
Ein gute Idee. Fast so leicht zu benutzen wie eine Telefonbuch. Es gibt davon ebenfalls eine gedruckte Ausgabe.

The Spider Web's Hot Lists
Adresse: http://gagme.wwa.com/~boba/hot.html
Ein Verzeichnis anderer Verzeichnisse auf dem Web.

Starting Point
Adresse: http://www.stpt.com/
En sehr schnelles und einfach zu benutzendes Verzeichnis mit 9 Kategorien. Täglich gibt es einen ausgesuchten Link.

Tom Duffys WWW Links
Adresse: http://www.stx.com/~tduffey/
Wiederum ein Verzeichnis mit zahlreichen Kategorien. Wenig Graphiken.

Top of the Web Home Page
Adresse:http://www.ibic.com/IBIC/TopOfTheWeb.html
Eine Zusammenfassung mit Links von anderen Verzeichnissen. Nur das Beste wurde ausgewählt.

XPlore
Adresse: http://www.xplore.com/xplore500/index.html
Verschiedene Versionen für verschiedene Verbindungsgeschwindigkeiten. Sehr gut organisiert und ein spezieller Link täglich.

Deutsches Yahoo

Die deutsche Version eines der größten Site Verzeichnisse
