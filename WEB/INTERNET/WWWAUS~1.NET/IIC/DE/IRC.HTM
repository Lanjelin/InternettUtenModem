1. WAS KANN ICH DAMIT TUN?

IRC ist die Abkürzung für Internet Relay Chat. Dieser Dienst ist wohl der, der am meisten Spaß macht. Es handelt sich dabei um eine Online-Diskussion (eigentlich währe Online-Tratsch ein besseres Wort) die in verschiedenen Channels abläuft. Sie können sich mit Personen auf der ganzen Welt privat oder in aller Öffentlichkeit unterhalten.

2. WAS BENÖTIGE ICH DAFÜR ?

4. WIE KANN ICH IRC VERWENDEN ?

Bevor wir jedoch beginnen, einige Worte über das IRC selbst. Das erste das Sie wissen sollten ist, daß IRC ein Kommandozeilen orrientierter Dienst ist (auch wenn Sie einen Window basierten Client verwenden). Um an einem Chat teilnehmen zu können müssen Sie in einen Channel hineingehen. Kein IRC Teilnehmer verwendet seinen richtigen Namen. Jeder benutzt einen Nickname. Das macht das ganze etwas einfacher und auch etwas spaßiger.

BEMERKUNG: Der von Ihnen gewählte Nickname gehört jedoch nicht Ihnen. Jeder Andere kann diesen Nickname verwenden, wenn Sie gerade nicht im IRC sind. Allerdings, wenn ein Nickname bereits verwendet wird, dann muß er eindeutig sein. Bekannte Nicknames werden also häufig von verschiedenen Personen verwendet, was leicht zur Verwirrung führen kann.

Um einen Nickname zu wählen, suchen Sie ein Optionsmenü in Ihrem IRC Client, indem Sie Ihren Nickname angeben können, oder verwenden Sie den folgenden Befehl:

/nick <name>

wobei <name> durch den gewählten Nickname ersetzt wird. Wie bereits erwähnt kann dieser Nickname bereits in Verwendung sein, dann haben Sie keine andere Möglichkeit als einen anderen Nickname zu wählen. Ein Beispiel:

/nick Bihuu

Alle anderen Benutzer sehen dann folgende Meldung:

* <oldnick >is now known as Bihuu

wobei <oldnick > Ihr vorheriger Nickname (das kann entweder Ihr Login-Name oder ein vom System gewählter Name sein) ist.

DIE CHANNELS
Wie bereits vorher erwähnt, besteht das IRC aus mehreren Channels. Jeder dieser Channels hat einen Namen. Bevor Sie in einen Channel wechseln, möchten Sie vielleicht eine Liste aller verfügbaren Channels sehen. Dazu verwenden Sie den Befehl:

/list

BEMERKUNG: Alle IRC Befehle beginnen mit einem vorangestellten Slash ("/"). Wenn Sie dieses Slash vergessen, dann wird der Befehl als normale Mitteilung behandelt und alle IRC Benutzer im selben Channel können diesen sehen.

Den Befehl den wir gerade verwendet haben wird eine sehr lange Liste ausgeben, da dieser Befehl jeden Channel listet, egal ob sich Personen darin befinden oder nicht. Um die List auf alle Channels mit mindestens 20 Benutzer zu beschränken, verwenden Sie den folgenden Befehl:

/list -min 20

Die nun ausgegebene List ist deutlich kürzer. Wenn sofort in einen Channel wechseln möchten, hier sind einige Channels, die meistens sehr gut besucht sind:

#ChannelHomepageThema
#insanity #Insanity Verrückter Chat
#prog #Prog Alles über das PRogrammieren
#StarWars #StarWars Möge die Macht mit Ihnen sein
#StarTrek #StarTrek Sehr unlogisch !
#talk #Talk Naja ...
#talk2me #Talk2Me Naja ...
#love #Love Wahre Liebe
#Windows95 #Windows95 Alles üer Windows 95

Ein weiterer guter Tip ist zu versuchen, daß Sie einen Channel finden, der genauso heißt wie das Land in dem Sie wohnen. Wenn Sie also in Deutschland wohnen, dann versuchen Sie einmal den Channel #germany. Wenn Sie in einen Channel wechseln, der noch gar nicht existiert, dann wird dieser erstellt, sobald Sie hineingehen. Sie bekommen dann automatisch die Rechte als Channeloperator, da Sie diesen Channel erstellt haben.

IN DEN CHANNEL WECHSELN
Wenn Sie einen Channel gefunden haben, in den Sie hineingehen möchten, dann erreichen Sie dies mit dem Befehl

/join <channel>

Wobei <channel> der Name des Channels ist, in den Sie wechseln möchten.

BEMERKUNG: Channelnamen beginnen immer mit einem "#"

Sobald Sie in einen Channel hineingegangen sind, wird das Thema des Channels und eine Liste der Personen (Nicknames) auf diesem Channel angezeigt. Unmittelbar danach sehen Sie wahrscheinlich schon die ersten Mitteilungen der anderen Benutzer.

DEN CHANNEL VERLASSEN
Da verschiedene Clients es dem Benutzer erlauben auf mehreren Channels präsent zu sein, müssen Sie beim Verlassen eines Channels angeben, welchen Channel Sie verlassen möchten. Das geschieht mit dem folgendem Befehl:

/leave <channel>

Wiederum gibt <channel> den Namen des Channels an, den Sie verlassen möchten.

TRATSCHEN
Nun können wir mit dem Chatten beginnen. Während Ihrer Unterhaltung werden Sie mit drei verschiedenen Arten von Mitteilungen konfrontiert werden. Die erste Art sind die Server Messages. Diese Nachrichten werden nicht von einem Benutzer geschrieben, sondern vom IRC Server. Solche Nachrichten werden normalerweise generiert, wenn ein neuer Benutzer den Channel betritt oder verlässt oder wenn irgendein Statuswechsel auftitt. Hier einige Beispiele von Server Messages;

* Jeannie (bottle@master.com) has joined channel #ghost
* Signoff Kahn
* You have been invited by NiceGirl to channel #talk

Server Messages beginnen normalerweise mit einem oder drei "*".

Die zweite Nachrichtenart sind persönliche Nachrichten. Im Gegensatz zu einer öffentlichen Nachricht, die von allen Benutzern gesehen werden kann, sieht eine private Nachricht nur der, der Sie auch sehen soll. Wenn Sie eine private Nachricht erhalten haben, so können Sie das am Nickname des Senders, der von einem "*" vorne und hinten eingeschlossen ist, erkennen. Hier ein Beispiel:

*JaneDoe* Hey ! Why don't you come over ?

Wenn Sie selber eine private Nachricht verschicken wmöchten, dann verwenden Sie dazu den /msg Befehl wie hier demonstriert:

/msg JaneDoe Surly I'll come over

Wenn der Nickname an den Sie eine Nachricht schicken möchten nicht existiert, dann bekommen Sie eine Fehlermeldung, die Ihnen folgendes mitteilt: no such nickname/channel. JaneDoe muß natürlich durch den Nickname der Person, an die Sie eine Nachricht schicken möchten ersetzt werden.

Alle anderen Nachrichten sind öffentliche Nachrichten. Diese sind daran zu erkennen, daß der Nickname des Senders in spitze Klammern eingeschlossen ist. Hier ein Beispiel:

<JaneDoe> Great !

Es gibt allerdings noch einige interessante Befehle, mit denen das Chatten noch mehr Spaß macht. Diese Befehle werden im nächsten Kapitel behandelt.

SPEZIELLE BEFEHLE
Wenn Sie detailierte Informationen über die Person hinter dem Nickname erfahren möchten, dann können Sie dies mit dem Befehl

/whois <nickname>

tun. Wiederum wird <nickname> mit dem Nickname der Person ersetzt, über die Sie Informationen einziehen möchten.

Wenn Sie ganz kurz Ihren Bildschirm verlassen müssen und das den anderen Teilnehmern mitteilen möchten, dann verwenden Sie dafür den Befehl

/away <reason>

Wobei <reason> ein Text sein kann, der angezeigt werden soll, damit die anderen Teilnehmer wissen, daß Sie kurzzeitig nicht erreichbar sind. Wenn Sie diesen Befehl eintippen, dann sehen alle anderen Benutzer die folgende Server Message:

* nick is away (reason).

wobei fü nick automatisch Ihr Nickname eingesetzt wird.

Wenn Sie ein Benutzer immer mit privaten Nachrichten bombardiert, und Sie das ganze absolut nicht interessiert, dann können Sie diesen Benutzer ignorieren, indem Sie den folgenden Befehl verwenden:

/ignore <nick>

wobei nick mit dem Nickname der Person zu ersetzten ist, die ignoriert werden soll. Sie bekommen dann von dieser Person keine (persönlichen) Nachrichten mehr.

Wenn Sie eine Person auf einen bestimmten Kanal einladen möchten, dann verwenden Sie den Befehl:

/invite <nick> <channel>

Wiederum muß <nick> mit dem Nickname der Person ersetzt werden, die Sie einladen möchten. <channel> Ist der Name des Channels auf den Sie den Benutzer einladen möchten. Die Person die Sie mit diesem Befehl einladen sieht dann folgendes auf dem Bildschirm:

* You have been invited to channel <name> by <nick>

Wenn Sie irgendeiner Aktion besonderen Ausdruck geben möchten, dann können Sie dafür den Befehl

/me <message>

verwenden; wobei <message> eine beliebige Nachricht ist, die angezeigt werden soll. Alle anderen Benutzer sehen dann eine Nachricht die so aussieht:

* nick <message>

Nick wird automatisch mit Ihrem Nickname ersetzt.

Wenn Ihnen eine Person so lästig wird, daß Sie diese am liebsten aus dem Kanal schmeißen würden, dann kicken Sie Ihn mit dem folgenden Befehl hinaus:

/kick <channel> <nick>

Wobei <channel> durch den Namen des Channels aus dem der Benutzer gekickt werden soll und <nick> mit dem Nickname des Benutzers der gekickt werden soll zu ersetzten ist. Wie bereits erwähnt hat jeder Channel ein bestimmtes Thema. Dieses Thema kann mit dem Befehl

/topic <channel> <topic>

geändert werden. Die Parameter dieses Befehls sollten in der Zwischenzeit selbsterklärend sein.

Wenn Sie in einen Channel hineingehen, dann sind Sie ein normaler Benutzer. Es gibt allerdings auch spezielle Benutzer, die Operators. Diese Personen haben zusätzliche Rechte. Sie können den Mode für andere Benutzer ändern, das Channel Thema ändern und vieles mehr. Es gibt eine ganze Reihe von verschiedenen Modi, die einem Benutzer verliehen werden können. Als Beispiel möchten wir zeigen, wie man einm Benutzer den Operator Modus verleiht:

/mode <nick> +o

Alle anderen Benutzer auf dem Channel sehen dann:

* Mode change: +o <nick> by <nick>

Wenn Sie einer Person den Operator Status wieder wegnehmen möchten, dann ersetzen Sie im obigen Befehl das "+o" durch ein "-o". Wenn Sie wissen möchten ob ein Benutzer ein Operator ist, dann verwenden Sie den /whois Befehl (siehe vorher). Befindet sich ein "@" vor dem Namen des Benutzers, dann handelt es sich um einen Operator.

BEMERKUNG: Einige Befehle können nur von Operatoren ausgeführt werden. So zum Beispiel das Ändern des Themas oder das Ändern des Modus für andere Benutzer.

NOTE: Ein Operator Status gilt nur für den Channel, auf dem er vergeben wurde. Beim freiwilligen oder erzwungenen verlassen des Channels geht der OP Status verloren.

5. ZUSÜTZLICHE INFORMATION

Auf manchen Channels finden Sie sogenannte Bots. Diese kleinen Programme sehen nach außen hin wie ein regulärer IRC Benutzer aus. Es handelt sich jedoch dabei um Programme, die jede öffentliche Nachricht eines Benutzers zum Beispiel auf bestimmte anstößige Worte hin überprüft, und den entsprechenden Benutzer dann aus dem Channel kickt. Weitere Beispiele sind Bots, die das Flodden eines channels bestrafen. Bots lassen sich teilweise erkennen, wenn man in einem Nickname das kleine Wort Bot findet. Generell eine gute idee, werden leider Bots auch häfig zu negativen Zwecken eingesetzt,
