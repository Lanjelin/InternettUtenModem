Link auf Ihre Site !
25. Jänner 1997

Wenn Sie auf Ihrer Seite einen Link auf das IIC eingebaut haben, dann können Sie einen Link auf Ihre Site von hier erhalten. Um diesen Link zu erhalten, füllen Sie das Formular aus und klicken Sie auf Abschicken. Wir werden uns dann Ihre Site ansehen und einen Link auf Ihre Site hier einfügen. Wenn Sie eine Site betreiben, die in das Konzept des IIC paßt, dann linken wir Ihre Site auch von anderen Teilen des IIC. Verwenden Sie dazu bitte das Site Submission Formular. Sie können auch eine Mail an an uns schicken.
