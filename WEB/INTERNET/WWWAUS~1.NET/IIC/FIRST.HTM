First Visit
January, 11. 1996

Welcome to the Internet Information Center. The purpose of this site is, to inform you about the Internet itself and how to use various different services, the internet has to offer. You will find everything necessary to make the Internet work for you. The purpose of this document is, to introduce you to the navigation provided for this site. To get the best out of this site, I recommend either Microsoft Internet Explorer 3.0 or Netscape Navigator 3.0. or any other browser that is capable of

  • displaying frames
  • executing javascript

NOTE: There is a non-frame version of this site available, but you will not have full navigation control. Since the web is always under development, we recommend to download one of the browsers mentioned above, which are available for a variety of platforms.

THE MAIN LAYOUT

If your browser supports frames, then you will see three frames in your window. The top frame is the main toolbar, the frame to the left is the menu bar and the frame in which this document is displayed is called the content frame which displays the desired documents. If your browser can execute javascript, then a short explanation will be displayed in your browsers status line.

NOTE: If you do not see eight buttons in the main toolbar, then please resize your browser horizontally.

DOCUMENT STRUCTURE

All the documents on this site have been designed with the ease of use in mind. They are separated in chapters and at the end of each chapter you will find a special symbol, which will take you to the top of the document (see symbols for details). Especially the documents in the how to section are structured the following way:

  • 1. WHAT CAN I DO ?
  • 2. WHAT DO I NEED ?
  • 3. WHERE CAN I CONNECT ?
  • 4. HOW DO I... ?
  • 5. ADDITIONAL DOWNLOADS
  • 6. ADDITIONAL INFORMATION

Furthermore topics in any document are formatted using this color.

NOTE: Due to the speciality of some services, not all of the above chapters might be available and/or there might be some additional chapters if required.

THE MAIN TOOLBAR

The main toolbar is located at the top of your browser window and provides buttons, that, if clicked, will take you to the

  • what's new section - where you will see what has been added, updated or removed.
  • first visit section - which you are currently reading.
  • feedback form - where you can send us your comments regarding the IIC
  • guestbook - Let the world know you where here and see who already has been here
  • advertising info - We are offering advertising programs at low rates.
  • german version - If you are from Austria, Germany, Switzerland, you might want to read the IIC in german.

THE MENU BAR

The menu bar is located to the left and allows you to navigate the different chapters and documents the IIC has to offer. From the main menu you can access the

  • netintro chapter - which introduces technical aspects and how to behave on the net
  • howto chapter - how to use different services and to accomplish certain tasks
  • more info - additional references and information
  • links - various interesting links to site directories, search engines, ...
  • glossary - where internet terms are explained clearly

INFORMATIONAL FOOTER

The document footer is located in the lower right corner of each document. It contains the copyright information and a link to the IIC's supporting internet provider EUnet.
