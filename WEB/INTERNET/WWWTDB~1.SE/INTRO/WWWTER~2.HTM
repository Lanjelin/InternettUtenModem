WWW - Terminologi

För att vi enklare skall förstå World Wide Web börjar vi med att definiera några av de termer som ofta förkippas med WWW och Internet.
WWW, W3
World Wide Web.
Hemsida
Huvudsidan för en person, företag eller organisation.
Personlig hemsida
Ett dokument (en sida) där vi beskriver oss själva.
Hyperlänk
Markerad text, ofta i annan färg eller understruken. Genom att klicka på en hyperlänk så öppnas och visas det dokument som länken refererar till. Detta dokument kan finnas varsomhelst i världen.
Hypertext
En text innehållande hyperlänkar.
HTML
HTML-HyperText Markup Language. Det språk i vilket dokumenten i World Wide Web är skapade.
HTTP
HTTP-HyperText Transfer Protocol. Det protokoll som används för att skicka "Web"-sidor över nätet, d.v.s. det språk datorerna pratar med varandra. Det är därför alla adresser börjar med http:.
URL
URL-Uniform Resource Locator. En URL är en referens till något på Internet. En URL består av två delar: det protokoll som krävs för att utnyttja tillämpningen samt dess adress.

protokoll://adress
Ett exempel: http://www.tdb.uu.se/research.index.html
FAQ
FAQ-Frequently Asked Questions. Ofta finns FAQs som sammanställer vanliga frågeställningar kring olika ämnen.
