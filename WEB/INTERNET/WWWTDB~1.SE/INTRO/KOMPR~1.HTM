Komprimerade filer

Dokument i FTP-arkiv ligger ofta komprimerade för att minimera minnesutrymmet och för att det skall gå snabbt att skicka dokumenten över Internet. På dokumentsuffixet så kan man se vilket komprimeringsprogram som använts.

De vanligaste suffixen för komprimerade filer är

.zip
pkzip är komprimeringsprogrammet, vanligt för PC-filer. Packas upp med pkunzip <filnamn>.
.gz
gnu-zip är komprimeringsprogrammet, vanligt för UNIX-filer. Packas upp med gunzip <filnamn>.
.Z
komprimerad med UNIX compress-kommando. Packas upp med uncompress <filnamn>.
.tar
arkiverad fil med UNIX tar-kommando. Förekommer ofta tillsammans med .Z och .gz. Packas upp med tar xvf <filnamn>.
.sit
Macintosh-format, komprimeringsprogrammet heter SuffIt.
Om man är ovan med att komprimera/packa eller dekomprimera/packa upp dokument så kan det ibland ställa till problem.
