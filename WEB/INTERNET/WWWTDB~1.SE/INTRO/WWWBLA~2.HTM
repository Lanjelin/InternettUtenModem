WWW - bläddrare

För att "surfa på nätet" som det populärt kallas så behöver man en WWW-"browser" eller bläddrare som vi föredrar att kalla det. Ibland kallas det också för WWW-klienter. Det finns idag en mängd olika bläddrare att använda.

Vill du veta mer om olika bläddrare, använd länkarna nedan som leder till WWW-sidor där mer information om resp. produkt finns.

Netscape Navigator
Netscape Navigator är den idag överlägset mest använda produkten för att söka sig runt på nätet.
NCSA Mosaic
NCSA Mosaic var den första bläddraren som blev riktigt populär. Sedan april 1993 distribueras den gratis på Internet.
HotJava
Java är ett nytt objektorienterat programmeringsspråk speciellt utvecklat för Internettillämpningar. HotJava-bläddraren utvecklades tillsammans med språket.
Internet Explorer
Internet Explorer är Microsofts produkt för att "surfa" på nätet.
Förutom dessa finns en mängd andra nätbläddrare. En som kanske skall nämnas är Lynx, som är helt textbaserad.
