Knappar i Netscape Navigator

Det bästa sättet att lära sig hur World Wide Web fungerar är att verkligen använda det. Vartefter de olika möjligheterna beskrivs så bör dessa utföras.

Knappar i Netscape (verktygsraden)

Orden sida och dokument används synonymt i förklaringarna. Genom att klicka på en knapp i menyn utförs följande:

Back
Tar dig tillbaka till den sidan du var innan denna.
Forward
Flyttar dig framåt. Om du gör Back och sedan Forward kommer du tillbaka dit du började.
Home
Tar dig till den startsida som du har definierat.
Reload
Hämtar pånytt det dokument som visas. Bra då man skapar egna dokument, för att se t.ex. om förändringar man gjort visas. Om man definierat minnescache i Options-menyn så måste Shift-tangenten hållas nedtryckt samtidigt som man klickar på Reload-knappen.
Images
Om man valt att Netscape Navigator inte ska visa bilder (Auto Load Images i Options-menyn), kan man ändå välja att titta på dessa genom att klicka på knappen.
Open
Om man vill öppna ett dokument som man vet URL:en till.
Print
Skriver ut dokumentet på en skrivare.
Find
Möjlighet att söka efter ord i dokumentet som visas.
Stop
Ibland kan det ta lång tid att öppna ett dokument, vilket ofta beror på att nätet är väldigt belastat. Genom att klicka på Stop så avbryts hämtandet av dokumentet.
Övningar
  1. Klicka på Home-knappen. Visas den hemsida som definierats?
  2. Gå till Uppsala universitets hemsida, URL:en är http://www.uu.se/
  3. Gå till hemsidan för Institutionen för teknisk databehandling. Sök efter ordet kurser i dokumentet. Är det en hyperlänk? Se efter vad dokumentet innehåller.
