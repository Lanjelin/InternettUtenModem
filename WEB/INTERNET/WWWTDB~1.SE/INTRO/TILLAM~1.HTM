Tillämpningar på Internet

De ursprungliga tillämpningarna på Internet, d.v.s det man kunde utföra var egentligen:
email
Ett sätt att skicka meddelanden mellan användare på Internet. Fungerar i princip som vanlig post, men går oerhört mycket snabbare och är billigt.
telnet
Används för att utnyttja datorer som står på en annan plats i världen.
FTP
FTP -File Transfer Protocol. Ett sätt att skicka filer (dokument) mellan datorer på Internet.

Att använda email, telnet och FTP var tidigare inte helt enkelt, det krävdes nästan en särskild kurs för att behärska den programvara som utförde tjänsterna.

Idag då alla använder World Wide Web så har dessa tillämpningar integrerats i "Webben" med ett användarvänligt gränssnitt, vilket förenklat användningen avsevärt.

Elektronisk post eller email har med all säkerhet varit den mest använda tillämpningen på Internet. Med endast några få tangenttryckningar är det möjligt att skicka email nästan överallt i världen.

Telnet som är den andra ursprungsapplikationen för internet används för att komma åt avlägsna datorer, att kunna använda dessa på ett sånt sätt att det verkar som om man fysiskt satt intill datorn.

FTP (File Transfer Protocol) är ett Internet-protokoll som bygger på TCP/IP som används för att flytta filer mellan datorer på Internet. Dessa filer kan innehålla allt från enkla textmeddelanden till musik eller videofilmer.

Andra Internet-tillämpningar som tas upp är Archie, Gopher, Veronica. Dessa finns integrerade i WWW, precis som News, Internets diskussionsforum.
