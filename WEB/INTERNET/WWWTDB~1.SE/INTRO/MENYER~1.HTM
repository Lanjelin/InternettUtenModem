Menyer i Netscape Navigator

File
Möjlighet att öppna nya Netscape-fönster, skicka e-post, öppna dokument, skriva ut dokument, avsluta Netscape.
Edit
Klippa, klistra textmassor från dokument, sökning av text i dokument.
View
Valet Document Source i menyn är bra för att undersöka hur den WWW-sidan man har öppnad är skriven i språket HTML.
Go
Här finns listat alla de senaste sidorna du tittat på. Bra för att snabbt komma tillbaka där man var tidigare.
Bookmarks
Valet Add Bookmark lagrar adressen till det dokument du nu tittar på. Alla dina "bokmärken" visas i menyn, bra för att hitta tillbaka till intressanta sidor.
Options
Används för att definiera Netscape på det sättet som du vill ha det.
Directory
Menyvalen motsvarar i stort sett de knappar som finns under Location:-rutan.
Windows
För Netscape mail, news samt för avancerad hantering av adressregister och "Bokmärken".
Help
Ger hjälp.
Övningar
  1. Öppna en ny WWW-bläddrare. Utför övningarna i en bläddrare, läs instruktioner i den andra.
  2. Titta på HTML-koden för denna WWW-sida.
  3. Gå tillbaka till Uppsala universitets hemsida, gör ett bokmärke till den.
