Australiens Grand Prix - Racerapport (10. marts 1996)

AUSTRALIENS GRAND PRIX
RACERAPPORT
10-3-96

Warm-Up omgangen :
- Lamy startede fra pitten.
- Frentzen udgår med mekanisk fejl.

Starten :
- Villeneuve laver en perfekt start.
- I sving nummer 2 skrider Hill ud og begge Ferrari kørerne kommer forbi ham.
- Efter cirka en halv omgang er der en del zig-zag kørsel nede bag i feltet som resulterer i at Brundle rammer Herbert, Brundle ryger om "på ryggen" og ind i kattegruset hvor hans vogn brækker midt over.
- Brundle kommer ud af vognen og er er uskadt.

Løbet stoppet :
- Doktor Sid Watkins giver Brundle lov til at starte igen.

2. Warm-Up omgang :
- Alle komme fint rundt, incl. Frentzen.
- Brundle og Coulthard skal starte fra pitten.

2. start :
- Alle kommer fint fra start denne gang.

1. omgang :
- Stillingen i toppen Villeneuve, Hill, Irvine, Schumacher og Alesi.
- Brundle spinner og udgår.
- Schumacher kommer forbi Irvine.
2. omgang :
- Hurtigste omgang til Villeneuve 1:37,036.
3. omgang :
- Hurtigste omgang til Schumacher 1:35,759.
4. omgang :
- Hurtigste omgang til Schumacher 1:35,467.
- Rosset og Lamy havde et mindre sammenstød. Og Rosset lavede en imponerende burnout i sit forsøg på at få vogne til at vende i den rigtige retning igen.
5. omgang :
- Hurtigste omgang til Villeneuve 1:35,223.
6. omgang :
- Hurtigste omgang til Hill 1:34,893.
9. omgang :
- Hurtigste omgang til Villeneuve 1:34,781.
- Alesi er lidt for overoptimistisk i et forsøg på at overhale Irvine, skrider ind i ham og beskadiger sit side stykke samt den ene køler og må udgå, ingen skade på Irvine's vogn.
11. omgang :
- Hurtigste omgang til Villeneuve 1:34,290.
- Stillingen : Villeneuve -> 1.020 -> Hill -> 1.309 -> Schumacher -> 8,364 -> Irvine -> 13,117 -> Hakkinen.
14. omgang :
- Hurtigste omgang til Villeneuve 1:34,166.
15. omgang :
- Verstappen spinder af banen og udgår.
16. omgang :
- Hurtigste omgang til Villeneuve 1:34,157
- Salo i pit 14,7 sek.
17. omgang :
- Hurtigste omgang til Villeneuve 1:34,069.
18. omgang :
- Hurtigste omgang til Schumacher 1:33,651.
19. omgang :
- Schumacher i pit 12,9 sek.
20. omgang :
- Villeneuve indhenter de første med en omgang.
- Stillingen : Villeneuve -> 0,696 -> Hill -> 18,090 -> Irvine -> 12,732 -> Schumacher -> 10,778 -> Hakkinen.
21. omgang :
- Irvine i pit 11,7 sek.
24. omgang :
- Berger i pit 9,1 sek.
25. omgang :
- Stillingen : Villeneuve, Hill, Schumacher, Hakkinen, Barrichello.
- Coulthard udgår p.g.a. problemer med speedertrækket.
26. omgang :
- Barrichello i pit 11,3 sek.
- Hurtigste omgang til Villeneuve 1:33,633.
- Stillingen : Villeneuve -> 1,099 -> Hill -> 30,791 -> Schumacher -> 18,119 -> Hakkinen -> 4,220 -> Irvine.
27. omgang :
- Hurtigste omgang til Villeneuve 1:33,421.
29. omgang :
- Villeneuve i pit 17,6 sek. (problem ved det ene baghjul).
- Hill er nu foran Villeneuve.
- Barrichello udgår p.g.a. motorproblemer.
30. omgang :
- Hakkinen i pit 14,2 sek.
- Stillingen : Hill -> 38,349 -> Villeneuve -> 13,015 -> Schumacher.
31. omgang :

- Villeneuve overhaler Hill.
- Schumacher måtte en tur i kattegruset på grund af problemer med for bremserne.
- Schumacher i pit 1:03,200 rep. ved bremsecylinderen.
32. omgang :
- Schumacher i pit igen, og der blev han.
- Fisichella udgår p.g.a. koblingsproblemer.

- Stillingen : Villeneuve, Hill, Irvine, Berger, Hakkinen.

44. omgang :

58. omgang :
-Hill får det ternede flag som vinder af det Australske Grand Prix 1996.

Resultatet af det Australske Grand Prix 1996,
på de point givende pladser :

1 - Hill.
2 - Villeneuve.
3 - Irvine.
4 - Berger.
5 - Hakkinen.
6 - Salo.

Lidt personlige komentarer :

hal :
- At Hill i virkeligheden ikke havde fortjent sin sejr, ellers havde han kørt sig til den, men fik den serveret på grund af en mekanisk fejl hos sin holdkammerat.
- At Williams er væsentligt hurtigere end Ferrari ..... Den eneste grund til at Schumacher kunne følge med i starten, var at han kørte på væsentligt minder fuel, og havde planlagt et 2- eller 3-stop racr, hvor Williams-teamet, kun havde planlagt ey One-stop race ...

arne :
- At det er længe siden at der har været så godt gang i F1 som der var i dette løb.
- Det var en behagelig overraskelse at se der ikke var nogen "teamorder" i f.eks. Williams-teamet, d.v.s. indtil Villeneuve fil olieproblemer. Men som alle ved så går Frank Williams kun efter konstruktørernes VM og derfor skulle begge biler i mål, det var grunden til at Villeneuve ikke fik lov til at fortsætte for fulddamp.
