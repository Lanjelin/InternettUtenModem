Brasiliens Grand Prix - Racerapport (31. marts 1996)

BRASILIENS GRAND PRIX
RACERAPPORT
31-3-96

Vejret :
- Regnvejr før og under starten på løbet, derefter tørvejr og under løbets afslutning var det solskin.

Starten :
- Alle kommer godt fra start, imponerende i det føre.

1. omgang :
- Alle undtagen Tarso Marques klarer den første omgang.
- Stillingen : Hill, Villeneuve, Alesi, Barrichello, Schumacher
11. omgang :
- Barrichello har nogle fede fights med Alesi
15. omgang :
- Hill indhenter den bagerste vogn.
19. omgang :
- Verstappen i pit og kommer ikke ud igen.
22. omgang :
- Coulthard i pit - første mand ude på slicks
24. omgang :
- Schumacher i pit 11.5 sek. - ud på regndæk.
25. omgang :
- Hurtigste omgang til Coulthard
26. omgang :
- Rosset kører ind i pitmuren og totalskader sin racer
27. omgang :
- Villeneuve skrider ud i kattegruset efter en nærkamp med Alesi.
(nogle burde måske have fortalt Villeneuve at Alesi ikke flytter sig frivilligt).
28. omgang :
- Berger i pit og udgår.
28. omgang :
- Stillingen : Hill, Alesi, Barrichello, Frentzen, Hakkinen, Schumacher, Brundle
30. omgang :
- Coulthard smutter ud på den Brasilianske pampas og der bliver han.
- Hakkinen i pit 10.8 sek. - ud på regndæk.
- Brundle i pit 12.8 sek. - ud på regndæk.
31. omgang :
- Alesi haler ind på Hill
33. omgang :
- Barrichello i pit 10.0 sek. - ud på regndæk.
34. omgang :
- Stillingen : Hill, Alesi, Salo, Irvine, Frentzen
35. omgang :
- Irvine i pit 10.2 sek. - ud på regndæk.
38. omgang :
- Hurtigste omgang til Alesi - 1:35.148
39. omgang :
- Hurtigste omgang til Alesi - 1:35.132
- Hill i pit 11.2 sek. - ud på slicks
40. omgang :
- Schumacher i pit 8.4 sek. - ud på slicks
41. omgang :
- Alesi i pit 11.1 sek. - ud på slicks
42. omgang :
- Barrichello i pit 7.6 sek. - ud på slicks
43. omgang :
- Irvine i pit 5.8 sek. - ud på slicks
45. omgang :
-Hurtigste omgang til Alesi - 1:25.697.
46. omgang :
- Stillingen : Hill, Alesi, Schumacher, Barrichello, Hakkinen, Salo, Panis
- Hurtigste omgang til Hill - 1:24.731
48. omgang :
- Hurtigste omgang til Alesi - 1:23.701
49. omgang :
- Hurtigste omgang til Hill - 1:23.170
51. omgang : - Hurtigste omgang til Hill - 1:22.864
- Hurtigste omgang til Barrichello - 1:22.493
52. omgang :
- Hurtigste omgang til Alesi- 1:22.306
54. omgang :
- Hurtigste omgang til Hill - 1:22.060
54. omgang :
- Stillingen : Hill, Alesi, Schumacher, Barrichello, Hakkinen, Salo, Panis
- Nu kommer solen rigtigt frem og banen tørrer hurtigt !
56. omgang :
-Hurtigste omgang til Hill - 1:21.932
-Hurtigste omgang til Alesi - 1:21.884
57. omgang :
- Schumacher (2), Barrichello (3), Hill (1) i tæt formations kørsel.
60. omgang :
- Barrichello overhaler Schumacher, men Schumacher hurtigt forbi igen.
- 200 m længere fremme ryger Barrichello i kattegruset og udgår.
(for meget adrenalin og for lidt vejgreb).
62. omgang :
- Hill overhaler Schumacher med en omgang (surt show).
64. omgang :
- Hurtigste omgang til Hill - 1:21.623
65. omgang :
- Hurtigste omgang til Hill - 1:21.547
- Brundle som ligger på 7. pladsen spinder og udgår.
67. omgang :
- Kun 12 vogne tilbage.
71. omgang :
-Hill får det ternede flag som vinder af det Brasilianske Grand Prix 1996.

Resultatet af det Brasilianske Grand Prix 1996,
på de point givende pladser :

1 - Hill.
2 - Alesi.
3 - Schumacher.
4 - Hakkinen.
5 - Salo.
6 - Panis.
