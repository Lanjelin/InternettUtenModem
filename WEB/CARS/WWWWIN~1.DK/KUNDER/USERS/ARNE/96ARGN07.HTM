Argentinas Grand Prix - Løbet omgang for omgang - 7. april 1996
ARGENTINAS GRAND PRIX
LØBET OMGANG FOR OMGANG
7 - 4 - 96

Vejret :
Solskin.

Starten :
- Alle kommer godt fra start, undtagen Villeneuve der ryger tilbage på 10. pladsen.

1. omgang :
- Alle klarer den første omgang.
- Stillingen : Hill, Schumacher, Alesi, Berger, Coulthard, Barrichello.
3. omgang :
- Hurtigste omgang til Schumacher.
5. omgang :
- Villeneuve oppe på 6. pladsen.
6. omgang :
- Berger hurtigste omgang - 1:31,570.
- Hill Hurtigste omgang - 1:31,407.
7. omgang :
- Salo ude og køre "Rally-Cross", men kommer med igen.
10. omgang :
- Hill hurtigste omgang - 1:31,158.
- Villeneuve op på 5. pladsen.
11. omgang :
- Villeneuve hurtigste omgang - 1:31,093.
12. omgang :
- Alesi hurtigste omgang - 1:30,796.
- Verstappen af banen, men komme med igen tilbage på 11. pladsen.
17. omgang :
- Salo i pit for at komme af med noget af den ballast han havde samlet op på 7. omgang.
19. omgang :
- Hakkinen i pit og der bliver han (speeder problem).
21. omgang :
- Schumacher i pit - 7,1 sek. - ud på 4. pladsen.
22. omgang :
- Alesi i pit - 5,0 sek. - ud på 4. pladsen.
24. omgang :
- Hill i pit - 9,6 sek. - ud på 1. pladsen.
- Berger i pit - 7,9 sek.
25. omgang :
- Villeneuve i pit - 11,8 sek.
- Stillingen : Hill, Schumacher, Alesi, Villeneuve, Berger.
26. omgang :
- Badoa spinner af banen og vælter om på ryggen. Badoa kommer ud af vognen uskadt
- Safety Car kommer ud på banen og "samler feltet" efter sig.
30. omgang :
- Efter at Diniz havde været i pit og er kommet ud på banen igen, begynder vognen at at brænde, Diniz kommer uskadt ud af bilen og flammerne.
32. omgang :
- Safety Car kører ind i pitten igen, og løbet fortsætter.
- Frentzen kører ud i kattegruset og udgår.
- Stillingen : Hill, Schumacher, Alesi, Berger, Villeneuve, Coulthard, Verstappen, Irvine, Barrichello, Lamy, Herbert, Marques, Brundle, Panis, Montermini.
34. omgang :
- Schumacher's Ferrari begynder at udsende røgsignaler.
37. omgang :
- Hill hurtigste omgang - 1:30,116.
38. omgang :
- Hill hurtigste omgang - 1:29,847.
39. omgang :
- Schumacher i pit - 12,6 sek.
40. omgang :
- Lamy har fået nok for idag, og udgår.
41. omgang :
- Alesi hurtigste omgang - 1:29,711.
42. omgang :
- Alesi hurtigste omgang - 1:29,537.
44. omgang :
- Alesi i pit - 8,6 sek. - da han skal til at kører igen dør motoren (det tog 16 sek. at starte motoren igen) - ud på 9. pladsen.
45. omgang :
- Alesi overhaler Schumacher.
46. omgang :
- Schumacher i pit og der bliver han.
47. omgang :
- Berger i pit - 8,6 sek.
- Irvine i pit - 10,3 sek.
48. omgang :
- Salo udgår.
49. omgang :
- Villeneuve i pit - 7,8 sek.
- Stillingen : Hill, Berger, Villeneuve, Coulthard, Alesi, Barrichello.
50. omgang :
- Coulthard i pit - 7,2 sek. - ud på 6. pladsen.
51. omgang :
- Hill i pit - 7,7 sek. - ud på 1. pladsen.
56. omgang :
- Berger i pit - 6,2 sek. - ud på 4. pladsen.
57. omgang :
- Berger i kattegruset og udgår.
58. omgang :
- Hill's pitboard viser "PUMP OFF".
60. omgang :
- Verstappen kommer forbi Coulthard.
66. omgang :
- Alesi hurtigste omgang - 1:29,413.
72. omgang :
- Hill får det ternede flag som vinder af det Argentinske Grand Prix 1996.

Resultatet af Argentina's Grand Prix på de point givende pladser :

1: D. Hill
2: J. Villeneuve
3: J. Alesi
4: R. Barrichello
5: E. Irvine
6: J. Verstappen

Hurtigste omgang gik til J. Alesi - 1:29,413.
