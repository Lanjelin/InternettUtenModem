Europa Grand Prix - Løbet omgang for omgang - 28. april 1996
EUROPAS GRAND PRIX
LØBET OMGANG FOR OMGANG
28 - 4 - 96

Vejret :
Tørvejr, 15 grader C.

Starten :
- Coulthard laver en kanon start er er forrest et øjeblik, men bliver overhalet igen af Villeneuve.
- Hill og Berger får en dårlig start, alle andre kommer godt fra start

1. omgang :
- Alesi påkører Salo, efter igen at have været for overoptimistisk (ligesom i Australien), Alesi udgår, Salo kan fortsætte.
- De øvrige klarer den første omgang.
- Stillingen : Villeneuve, Coulthard, Barrichello, Schumacher, Hill, Hakkinen.
3. omgang :
- Hurtigste omgang til Villeneuve - 1:22,841.
- Berger i pit.
4. omgang :
- Hurtigste omgang til Berger - 1:22,496.
- Hill kommer forbi Schumacher.
5. omgang :
- Hurtigste omgang til Berger - 1:22,109.
- Stillingen : Villeneuve, Coulthard, Barrichello, Hill, Schumacher, Hakkinen.
6. omgang :
- Panis påkører Irvine, og de udgår begge.
8. omgang :
- Hurtigste omgang til Berger - 1:22,004.
10. omgang :
- Berger oppe på 16. pladsen.
16. omgang :
- Stillingen : Villeneuve -> 11,410 -> Coulthard, Barrichello, Hill, Schumacher, Hakkinen.
20. omgang :
- Frentzen i pit - 8,2 sek.
22. omgang :
- Hill i pit - 20,9 sek. (tilsyneladende problem med bagerste ophæng/affjedring), kommer ud på 11. pladsen.
23. omgang :
- Berger i pit - 9,4 sek.
- Stillingen : Villeneuve, Coulthard, Barrichello, Schumacher, Hakkinen.
24. omgang :
- Schumacher i pit - 7,7 sek.
- Barrichello i pit - 12,5 sek.
- Coulthard i pit - 9,8 sek.
- Villeneuve i pit - 8,9 sek.
26. omgang :
- Hill er på 10. pladsen efter sit pit besøg.
- Hurtigste omgang til Schumacher 1:21,976.
- Stillingen : Villeneuve -> 8,750 -> Schumacher, Hakkinen, Coulthard, Barrichello, Frentzen.
28. omgang :
- Diniz og Hill "gnider dæk" og ryger af banen, begge kommer dog tilbage på banen og fortsætter.
- Diniz i pit.
- Afstanden mellem Villeneuve og Schumacher 8,745 sek.
30. omgang :
- Hakkinen får en STOP'n'GO for at have kørt for hurtigi på vej ind i pitten.
32. omgang :
- Stillingen : Villeneuve -> 2,490 -> Schumacher, Coulthard, Barrichello, Frentzen.
36. omgang :
- Afstanden mellen Villeneuve og Schumacher nu kun 1.060 sek.
37. omgang :
- Stillingen : Villeneuve -> 0,660 -> Schumacher -> 12,970 -> Coulthard -> 3,840 -> Barrichello -> 7,200 -> Frentzen -> 2,190 -> Hakkinen.
39. omgang :
- Hurtigste omgang til Hill - 1:21,695.
42. omgang :
- Frentzen i pit - 7,8 sek.
42. omgang :
- Berger på 11. pladsen.
- Hill på 9. pladsen.
43. omgang :
- Schumacher i pit - 8,5 sek. kommer ud på 3. pladsen.
- Rosset i pit - 7,4 sek. DAGENS BEDSTE PITSTOP.
45. omgang :
- Villeneuve i pit - 8,3 sek. kommer ud på 1. pladsen.
- Schumacher overhaler Coulthard.
46. omgang :
- Brundle i pit - 8,6 sek.
- Stillingen : Villeneuve -> 2,680 -> Schumacher -> 3,920 -> Coulthard -> 3,230 -> Barrichello -> 6,440 -> Hakkinen -> 0,390 -> Hill.
48. omgang :
- Hakkinen får dagens anden STOP'n'GO for at komme for hurtigt ind i pitten.
51. omgang :
- Hurtigste omgang til Hill - 1:21,598.
52. omgang :
- Stillingen : Villeneuve -> 0,480 -> Schummacher, Coulthard, Barrichello, Hill.
- Hurtigste omgang til Hill - 1:21,463.
55. omgang :
- Hurtigste omgang tik Hill - 1:21,363.
60. omgang :
- Stillingen : Villeneuve -> Schumacher -> ??? -> Coulthard -> 1.180 -> Hill -> 1,710 -> Barrichello -> Brundle.
- Frentzen udgår efter en tur i kattegruset.
64. omgang :
- Nu kan Schumacher "lugte gearkasseolien" på Villeneuve.
- Hill helt oppe i r.... på Coulthard.
65. omgang :
- Afstanden mellem Villeneuve og Schumacher 0,210 sek.
67. omgang :
- Villeneuve vinder der Europæriske Grand Prix på Nürburg Ring 1996 med en margen på 0,762 over Schumacher -> 32,070 -> Coulthard -> 0,680 Hill -> 0,200 -> Barrichello -> 21,850 -> Brundle.
