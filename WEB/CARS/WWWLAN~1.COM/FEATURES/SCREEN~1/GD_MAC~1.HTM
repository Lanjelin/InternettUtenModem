Macintosh Screen Saver Instructions

  1. Download the Great Divide Screen Saver file:"great_divide_ss.hqx".
  2. If your browser is set up to use Stuffit Expander it will automatically de-binhex and unstuff the "great_divide_ss.hqx" file.
  3. If your browser is not set up to run Stuffit Expander automatically, you must run Stuffit Expander on the file.
    1. Run Stuffit Expander
    2. Select Expand from the File pulldown menu.
    3. Select the downloaded file "great_divide_ss.hqx" and click Expand. This will de-binhex and unstuff the file.
  4. Copy the Great Divide Screen Saver control panel to your System Folder and reboot.
  5. Double-click on the Great Divide Screen Saver control panel, to set your Screen Saver preferences.
