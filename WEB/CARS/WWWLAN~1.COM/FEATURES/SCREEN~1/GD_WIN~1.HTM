Windows 95 Screen Saver Instructions

  1. Download the Land Rover Great Divide Screen Saver from this location (you may get a dialog box indicating that you are about to download an unknown file type, choose the "save file" option) and save into a temporary directory.
  2. The file GD96inst.EXE is a compressed self-extracting installer. It must be decompressed on your hard drive before you can install the screen saver.
  3. To decompress the file that you just downloaded click the Start button. Select Run. Click Browse.
  4. Find and select the GD96inst.exe file. Click Open. Click OK.
  5. When the WinZip Self-Extractor dialog box appears, click the Unzip button. This will create a directory on your c:\ drive called GD96. This will also create a file within c:\GD96 called GD96.exe.
  6. The file will then be automatically installed.
  7. After the screen saver has been succesfully installed, you may set the Great Divide file as your screen saver by clicking on the Start button; highlighting Settings, and selecting Control Panel. Then double-click the Display icon.
  8. The Display dialog box will open. Select the Screen Saver tab.
  9. Select Great Divide '96 for your screen saver. Click Apply to keep Great Divide as your screen saver.
  10. Click OK to finish.
