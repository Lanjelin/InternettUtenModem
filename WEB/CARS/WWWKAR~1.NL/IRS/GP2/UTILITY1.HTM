 By Steven Young GP2Edit allows you to create single file car sets which contain all car textures, helmet textures, tyre textures (optional), performance data and much more... GP2Edit supports all game versions and should work on all Windows versions running in 256 or more colours.
More information about the program is available at http://ourworld.compuserve.com/homepages/smyoung/gp2edit.htm.
Utility "Mostra Assetto" v1.3 (22kb) By Andrea Longoni This utility displays the setup information in ITALIAN language from a single car setup file (.CSx) which you specify on the command line. You will be pleased to know that if you redirect the output to a file (e.g. assetto race.csh > race.txt), the setup information will be sent to the file.

Playtime logger v1.1a (25kb) By Petri Tuominen This is a program that keeps a log of time spent with GP2. As you probably know, GP2 tells you the time you spent with it after you exit. This program grabs that information and adds it to its own log and then prints the amount of TOTAL time spent with GP2. Don't get scared! :)
