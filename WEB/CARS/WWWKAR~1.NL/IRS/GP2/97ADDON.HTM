MasaYard's IRS Season 97 Add-on

Introduction

The season 1997 have begun !!! We made an add-on which will update your GP2 into season 1997 !!! The current version of this add-on will only replace the graphics, not the tracks. The add-on includes:

  • New cars
  • New helmets
  • New car and driver performances
  • New cockpit colours
  • New pitcrew colors
  • New pitbox order
  • New GoodYear tyres
  • New marshals and chequered flag man
  • New starting lights
  • Installation batch for DOS
  • Support for all versions of GP2

Installation

1. Create directory SEASON97 under your GP2-directory.
2. Unzip the ZIP-file in the directory.
3. Type INSTALL.
4. Run GP2 and start playing.
5. If you have any problems, check README.TXT for trouble shooting.


Update installation

1. Unzip the ZIP-file in the SEASON97 directory;
2. Type INSTALL.
3. Run GP2 and start playing.


Editor for Game Settings

You need Slobodan Blanusa's GP2 Editor v2.12b for Windows to change the game settings like quickrace lenght and performaces. Other options can be changes in the game menus.
