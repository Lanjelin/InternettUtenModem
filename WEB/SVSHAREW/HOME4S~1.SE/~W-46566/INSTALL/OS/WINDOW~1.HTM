Uppgradera och Installera Windows 95

Vill du ha ett modernare operativsystem, är du nybörjare eller tycker du att alla inställningar i Windows 3.11 är svåra att göra så bör du börja med eller uppgradera till Windows 95. Det hanterar många av inställningarna som du förut behövde ställa in helt automatiskt. De tre stora fördelarna med Windows 95 är:

  • Förbättrad multitasking
  • Långa filnamn
  • Enklare att växla mellan program

Du behöver inte oroa dig för att dina gamla Windowsprogram inte fungerar. Nästan alla gamla program fungerar bra. För Dos-program som spel finns det förutom den vanliga Dos-prompten ett speciellt läge där bara Dos programmet körs. Genom det läget kan man alltså få igång även dem Dos program som inte kan köras direkt under Windows. Den enda typen av program som inte bör användas är program som direkt ändrar i filsystemet, t.ex. gamla Undelete och Defrag och liknande program. Säkerhetskopieringsprogram fungerar men de kan inte använda de långa filnamnen, men det finns numera program som separat kan säkerhetskopiera dem långa filnamnen om du vill använda gamla säkerhetskopieringsprogram.

Det finns några bra tillbehörsprogram som du bör titta på om du har många gamla program. Först och främst är finns det program som ger gamla program möjlighet att använda långa filnamn, ett sådant ingår i Norton Navigator för Windows 95. Ett annat bra program är programmet DosLfnBk (gratis) som säkerhetskopierar dem långa filnamnen, vilket gör att du kan använda gamla säkerhetskopierings-program. Titta också gärna på olika sharewareprogram vilka som vanligt fyller många luckor. I mitt tycke så medföljer det inte tillräckligt många småprogram med Windows, men dem hittar du bland sharewareprogrammen.

Som sagt måste du ha nya program för att fullt utnyttja Windows 95. Dessa program ska vara så kallade 32-bitars program. Program utvecklade särskilt för Windows 95 fungerar bäst, men även en stor del av 32-bitarsprogrammen för Windows NT och alla Win32s fungerar bra.

En annan sak i Windows 95 är dock att det från början ser stökigare ut p.g.a. av att man kan ha mappar (f.d. kataloger) och filer direkt på skrivbordet, men efter att du har arbetat ett tag med operativsystemet kommer du att upptäcka att det är enklare än föregående Windows förutsatt att du har nya 32-bitars program som är speciellt anpassade för Windows 95, eftersom dem har samma funktioner som operativsystemet i övrigt, t.ex. går det att använda högra musknappen.

Särskilt dem långa filnamnen gör programmen enklare att använda eftersom du då kan ge filerna meningsfulla filnamn och även välja att spara viktiga filer direkt på skrivbordet så att du ser dem direkt nästa gång när du använder datorn.
