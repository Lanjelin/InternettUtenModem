Uppgradera och Installera Windows 95

Installationstips

  • Skapa en startbar diskett, med autoexec.bat och config.sys inställda så att drivrutinerna till din CD-Rom laddas från disketten. Använd sedan denna diskett för att starta när du ska uppgradera.
  • Kolla din hårddisk med Scandisk och kör Defrag, vilket du också bör göra med jämna mellanrum vid normal användning.
  • Ta en säkerhetskopia, om du behöver någon gammal inställning så kan du hämta den från säkerhetskopian.
  • Ta nu bort både din gamla Dos och Windows katalog, inklusive systemfilerna.
  • Starta sedan med startdisketten och börja installationen av Windows 95, har du köpt en uppgraderingsversion så "visar" du upp första disketten från din gamla Windowsversion vid installationen.
  • Ansluta sedan med att installera dina program och anpassa ditt system som du vill ha det.

Windows 95 är igång.
