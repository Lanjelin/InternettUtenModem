Att Rensa och ställa in Dos

Inställningar i startfilerna

Sökvägen

Ställ in sökvägen för din dator genom att ändra på raden i AutoExec.bat där det står ungefär:

Path C:\Dos

Genom att lägga till flera kataloger på raden kan du köra program i dessa kataloger genom att skriva programmets filnamn även om du inte befinner dig i programmets katalog. Alla programkataloger behöver du dock inte lägga till, bara dem där du har flera program, t.ex. Dos-tillbehör.

Din sökväg kan till exempel se ut så här:

Path C:\Dos;C:\Utils;C:\Utils\DosUtils

Kom ihåg semikolonet mellan leden.

Temporär-katalog

Skapa en katalog för tillfälliga filer. Normalt så brukar den heta Temp och vara direkt i roten på hårddisken. Lägg därefter till följande rad till AutoExec.bat

Set Temp=C:\Temp

Du bör tömma denna katalog om den innehåller filer med filändelsen .tmp då du inte kör några program, men gör inte det när du kör program.

Doskey

Lägg till Doskey i Autoexec.bat genom att skriva Doskey på en rad längst ner. Detta kommer att bespara dig många tangentnedtryckningar, då du kan bläddra tillbaka till gamla kommandon med piltangenterna.

Snabbare start

Du kan starta upp din dator snabbare genom att lägga till raden Switches=/F till början av Config.sys då visas inte texten Startar MS-Dos... ett par sekunder som tidigare. Även minnes testet som himem.sys gör kan stängas av för att starten ska gå ännu snabbare, lägg i så fall till parametern /Testmem:Off efter himem.sys i Config.sys.

För snabbaste start ska din Config.sys alltså börja så här:

SWITCHES=/F
DEVICE=C:\DOS\HIMEM.SYS /TESTMEM:OFF
