Vad är detta?

Från denna web-site får du många tips som förhoppningsvis kommer att hjälpa dig när du använder din dator. Dem flesta tipsen är för Windows, men en del kan användas oberoende av operativsystem. Jag förutsätter att du redan kan använda Windows någorlunda bra och att du förut har installerat program på vanligt sätt, det vill säga genom att använda standardinstallations-metoden. Denna web-site är alltså inte riktad till direkta nybörjare utan för personer som vill bli effektivare och lära sig mer.

Först hittar du information om hur du installerar Windows så att du får ut det mesta av din dators prestanda. Jag kommer dessutom i ta upp lite annan fakta om operativsystem som kan vara bra att känna till.

Du hittar också information om hur du installerar program så att din dator inte blir för rörig och att du sedan i efterhand kan underhålla programmen och avinstallera dem.

Dessutom hittar du massor med diverse tips om hur du allmänt arbetar effektivare med program och var du kan få tag på små billiga sharewareprogram som löser många brister i Windows. Till sist också några metoder att säkerhetskopiera alla dina program, inställningar och dokument.

Jag har försökt att ta upp sådana tips som du alltid kommer att ha nytta av istället för att ta upp tips om specifika ändringar av inställningar som kommer med all sannolikhet att ändras i kommande versioner av Windows.

Lycka till med "Trimmandet av din dator",
Aki Tuuliainen, September 1996.
