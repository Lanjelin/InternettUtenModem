Att tänka på vid installation av program

Andra installationstips

Vill du efter installationen ta bort eller lägga till några delar av programmet kan du i de större programmen köra installationsprogrammet igen. Då får du möjlighet att välja vilka delar som ska tas bort eller läggas till.

Ett annat sätt att installera en viss fil från installationsdisketterna är att packa upp den manuellt med Dos kommandot Expand. Det fungerar om installationsfilernas filändelse slutar med ett understrykningstecken, _. Du packar upp en fil genom att ange källfilen och till filen som parametrar till kommatdot Expand, tex för att packa upp filen Ctl3D.dll skriver du Expand A:\Ctl3D.dl_ C:\Windows\System\Ctl3D.dll.

Skulle Expand inte fungera kan du använda programmet Decomp som brukar finnas på första installationsdisketten för programmet.
