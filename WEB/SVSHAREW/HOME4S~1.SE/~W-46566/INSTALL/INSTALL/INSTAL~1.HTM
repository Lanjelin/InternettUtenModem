Att tänka på vid installation av program

Installera programmen

Oftast installerar du ett program genom att köra programmet Setup eller Install från disketten eller CD-Rom skivan som programmet kom på.

Du väljer naturligtvis att installera programmet till en lämplig katalog i din katalogstruktur istället för standard-katalogen.

Program som du hämtat från Internet eller bbs:er är ofta packade och måste packas upp innan installation. Småprogram kan sakna installationsprogram och då räcker det oftast med att kopiera programmet till en lämplig katalog.

Med stora program medföljer det massor med tillbehör och andra filer, då kan det vara bra att välja anpassad installation och välja att bara installera bara det man behöver för att spara hårddiskplats. Det funkar bra då bara en person använder datorn, men om flera använder det så kan det vara bättre att installera nästan allt för att vara säker på att allt som kommer att användas finns med.

Efter installation

Alla filer som installeras med program behövs inte, sådana filer är t.ex. textfiler med senaste information och extra informationsfiler. Efter att du har läst dessa filer kan du antingen ta bort dem eller flytta dem till någon annan katalog för senare användning.
