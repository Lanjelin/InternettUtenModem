Installation

Skapa en katalogtruktur

Hur ofta har du sett en dator med alla program installerade i roten på hårddisken? Antingen ägs den av en person som inte kan använda datorer tillräckligt bra eller så har personen allmänt dåligt ordningssinne. Visst går det bra att ha programmen i rotkatalogen än så länge då man bara har ett par program, men därefter blir det lätt en lång lista med kataloger direkt i roten.

Efter att du har installerat operativsystemet och innan du installerar ytterligare program bör du därför skapa en katalogstruktur på din hårddisk. Gör du en bra struktur från början kommer det att vara mycket lättare att hitta program och filer, att senare installera mer program och att göra ändringar i installerade program utan att det blir svårare att hitta.

Innan du skapar din katalogstruktur bör du tänka igenom hur du kommer att använda din dator, om någon annan använder den och hur ni vill organisera den.

Tänk också igenom hur många och vilka program du har, vilka program du kommer att skaffa i framtiden. Skapa sedan utifrån dessa faktorer en passande katalogstruktur. Här följer några exempel på olika katalogstrukturer för olika användningsområden. Du kan givetvis skapa en katalogtruktur som innehåller lite av alla dessa alternativ.

Ordna katalogerna efter program-områden och skapa passande dokument kataloger.

Passar bra för många program. Filer som du själv skapar med programmen lagras antingen i dokument-katalogerna eller i programkatalogerna.

Skapa en programkatalog och en eller flera dokument kataloger.

Passar bra för ett mindre antal program.

Skapa kataloger där olika användare lagrar sina program och dokument.

Kan passa bra om flera personer använder datorn.

I Windows 95 kan du använda programmet Utforskaren för att skapa kataloger.
I Windows 95 har katalogerna ett annat namn, nämligen mappar.
