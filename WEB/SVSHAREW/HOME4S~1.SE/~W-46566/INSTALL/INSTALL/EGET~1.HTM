Att avinstallera program

Med eget avinstallationsprogram

Enklast är det att avinstallera nya program, vilka ofta har ett eget avinstallationsprogram.

I Windows 95 startar du Kontrollpanelen och går in i Lägg till/ta bort program. Där markerar du det program som du vill avinstallera och klickar på Lägg till/ta bort.

Ett bra tips är att inte ta bort avinstallationsprogrammet när du rensar bort onödiga filer.
