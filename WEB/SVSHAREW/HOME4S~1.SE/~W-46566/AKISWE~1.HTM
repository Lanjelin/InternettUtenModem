Den Svenska Shareware Listan

Äntligen en lista med bara svenska sharewareprogram! Alla program kan laddas ner direkt från sidan. Har du gjort ett svenskt program kan du tipsa om det och få med det i listan.

 

Så installerar du programmen i din PC

Massor med information om hur du installerar och använder programmen i din PC så effektivt som möjligt.

 

SkolPack

Fem svenska skolprogram, ladda ner och pröva! Innehåller ett bra glosprogram, ett mattespel, ett frågesportsprogram och spelet hänga gubbe.
Memory - Memory är det klassiska spelet där det gäller att komma ihåg var korten ligger så att du hittar paren.

TheWords - The Words är ett lättanvänt program för att träna ord till exempel dina engelska glosor.
