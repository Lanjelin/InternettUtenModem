Memory's Hemsida

Information

Memory är det klassiska spelet där det gäller att komma ihåg var korten ligger så att du hittar paren. Denna version kan spelas av upp till 10 spelare i tur och ordning. Tre stycken "temor" på bilderna medföljer: Blandade bilder, data bilder och serie figurer.
Memory är gjord för Windows 95, NT 4.0 och högre. Programmet är Freeware, vilket innebär att det får användas gratis.

Tre svårighetsgrader finns: 28, 54 eller 108 kort!

Ladda ner Memory

1. Innan du kan köra Memory måste Visual Basic 4.0 32-bit Runtime filerna finnas installerade i din dator.
Klicka här för att ladda ner runtime filerna (1.2 MB) .
Kör ovanstående fil så installerar runtime-filerna åt dig.

2. Ladda ner Memory.
Klicka här för att ladda ner Memory (765 KB).
På samma sätt som i ovanstående fall kör du filen för att installera Memory.
