On this page you will find a number of common questions relating to the use of the LEGO TECHNIC CD-ROM. If you have questions concerning the LEGO TECHNIC CD-ROM, please send your mail to this address: Helpdesk@LEGO.com
Under any circumstances it is highly recommended that you read the printed manual and the ReadMe text found in the LEGOTECH folder on your harddrive.
Why is it that my LEGO TECHNIC CD-ROM
runs slowly on my computer?
The CD-ROM will NOT run well with virtual memory or disk doubler on the Mac - And it will run slowly with virtual memory in very active use on a PC. There is also a "Turbo Mode" which will turn off the help-texts etc, making the CD-ROM running a bit faster. To activate this, press the Control "Ctrl" button while clicking the "OK" button on the log-in screen. To turn off this mode, return to the Log-on and click on the "OK" button with the Control "Ctrl" key again.
Why is the diver not speaking Danish or
Dutch when I select one of these languages?
The Danish and Dutch language versions
only comes with English diver speech.
How come my name "disappears"
after I have typed it in on the log-in screen?
Remember to press RETURN to confirm your name in login screen. If you want to enter another user at the same time, it is necessary either go into the programme and out again, or to press the "Legal Terms" button.
