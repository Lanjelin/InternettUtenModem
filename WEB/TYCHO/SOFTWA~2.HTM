World Wide Web

Netscape
Netscape is the web browser of choice at AO. It is able to handle a number of different types of data: still images, sounds, movies, 3D objects, and more. You can test whether your Netscape on your machine can do all that by visiting the MIME test page. If you find something not working, please let www@astro.ku.dk know.

AmericaOnLines HTML editor, is available on the SGIs and on the Solaris hosts, as aolpress (or gnnpress).  Try it out!
