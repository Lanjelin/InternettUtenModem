Dienstübersicht
October, 17. 1996

Die folgende Übersicht soll Ihnen die populärsten Dienste des Internet näher bringen. Indem Sie auf den Titel klicken werden Sie sofort zur entsprechenden Anleitung im Kapitel Wie funktionierts gelangen.

World Wide Web (kurz: WWW)

Dieser Dients ist der bekannteste und auch für den Wachstum des Internets verantwortlich. Informationen werden wie ein Buch oder eine Zeitung präsentiert. Sie sehen Texte und Bilder und höhren Sounds. Sie benötigen dazu einen Web Browser. Die bekanntesten Browser sind wohl NSCA Mosaic und der Netscape Navigator und mittlerweile auch der Microsoft Internet Explorer. Mit dieser grafischen Benutzeroberfläche können Sie durch's Web "surfen", indem Sie auf die sogenannten Hyperlink anklicken. Auf diese Art springen Sie von Seite zu Seite und von Server zu Server ... und wenn Sie in Österreich gestartet sind, befinden Sie sich im Nu in Amerika, Rußlan oder sonst wo auf der Welt. Außerdem können Sie Dateien downloaden.

File Transfer Protocol (kurz: FTP)

Dieser Dienst wird verwendet um Dateien von einem Computer zu einem anderen zu transferieren. Sie benötigen ebenfalls einen speziellen Client für diesen Service. Es gibt im Internet eine Reihe von sogenannten Anonymous FTP Servern, die vor allem Shareware, Freeware, Treiber, etc. zur Verfügung stellen. Allerdings müßen Sie beachten, daß reine Textdateien im ASCII Modus (Word Dokumente gehören nicht dazu !!) und alle anderen Dateien im Binär Modus übertragen werden müssen.

Elektronische Mail (kurz: E-Mail)

Sie können E-Mail dazu verwenden, um mit Personen rund um die Welt billig und schnell zu komunizieren. Sie benötigen dazu einen E-Mail Account auf einem Mail Server und einen Mail Client. Wahlweise können Sie auch einen Unix Account zu senden und empfangen von E-Mails verwenden. Es gibt außerdem einen interessanten Dienst, der sich der E-Mail bedient. Es handelt sich dabei um die sogenannten Mailinlisten. Mailinglisten sind E-Mail basierte Diskussionsforen, jede Mailingliste für ein spezielles Gebiet. Dabei werden Diskussionsbeiträge zu einem speziellen Server geschickt, der diese dann wiederum an alle Teilnehmer weiterleitet. Wenn Sie noch nie an einer elektronischen Diskussion teilgenommen haben, dann empfehle ich, die Netetiquette zu lesen.

Gopher

Gopher is ein text-orrientierter Dienst. Die Informationen sind in einer hierarchischen Struktur abgelegt. Wie bei allen anderen Diensten benötigen Sie einen speziellen Clienten und die Adresse eines Gopher Servers. Sämtliche Informationen können auch von einm Gopher Server heruntergeladen werden.

Usenet Newsgruppen

Die Newsgruppen sind tausende Diskussionsforen über alle nur erdenklichen Themen. Die Newsgruppen werden hierarchisch verwaltet. Ein Beispiel für eine solche Newsgruppe wär alt.binaries.windows.shareware. Es gibt außerdem ene Newsgruppe mit dem Namen alt.binaries.mac.shareware. Man kann deutlich die hierarchische Anordnung erkennen. Sie benötigen Zugriff auf einen Newsserver und einen Newsreader. Newsserver sind normalerweise nicht allgemein zugänglich. Entweder benötigen Sie ein Username und ein Passwort oder der Zugriff wird nur von bestimmten Domänen aus erlaubt. Es gibt zwar einige öffentliche Newsserver, diese werden jedoch immer weniger und sind schwer zu finden. Beachten Sie außerdem, daß ein Newsserver 10000 Gruppen führen kann, ein anderer jedoch mehr als 23500. Sie selbst können auch Artikel in eine oder mehrere Newsgruppen posten. Dazu ist in den meisten Newsreadern ein kleiner Mailprogramm implementiert. Moderne Web Browser erlauben ebenfalls das lesen und posten in Newsgruppen

Unix Terminal / Telnet

Unix is ein Mehrbenutzer Multitasking Betriebssystem, welches häfig auf Workstations eingesetzt wird. Wenn Sie Zugang zu einem Unix Account haben, dann können Sie viele Dienste auch von hier aus verwenden. Noch vor einigen Jahren war das die einzige Möglichkeit das Internet zu benutzen. Unix ist ein Kommandozeilenorrientiertes Betriebssystem, obwohl es auch eine grafische Oberfläche gibt.

Internet Relay Chat (short: IRC)

Dieser Dienst, ist wohl der Beste. Es handelt sich dabei um tausende Online-Diskussionsforen. Online bedeutet hier, daß Sie mit anderen Teilnehmern quasi in Echtzeit kommunizieren können. Es gibt im IRC sogenannte Channels, jeder mit einem speziellen Thema. Jeder Teilnehmer hat einen sogenannten Nickname über den er ansprechbar ist. Achtung: Suchtgefahr !
