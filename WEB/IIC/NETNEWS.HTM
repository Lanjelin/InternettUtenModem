Internet News

Welcome to our new section, called Internet News. The purpose of this secton is to provide links to interesting and new technologies and news that are of interest to you, the Internet user. We won't cover in depth technical news, if they are not necessary for using the Internet. We are always checking the latest developments and improvements and will give you a short outline of the article here. If you are interested in knowing more about this, you will find a more button, which will take you to the full article, which is not from us here, but from other sites.

The second part of this section is called hotspot, where we will cover a special topic each month. This can either be a site review, introduction of new software or technologies or anything similar. At this point, your action is required. What topics would you like to be covered here ? Even you can write an article for the hotspot. Contact us for details

Internet Explorer 4.0 Preview delayed
Because of the recently discovered security risks, the Preview Beta of Internet Explorer 4.0 will be delayed until the end of march. Details can be found at:

  • Bugs delay IE 4.0 Beta (TechWeb)
  • IE delay doesn't stop Microsoft from touting browser features (InfoWorld)
  • Microsoft delays Explorer 4.0 beta (C-Net)

Internet News

Internet Explorer 3.0 security whole found and fixed
Recently a security whole in IE 3.0 was discovered. Microsoft has already posted a fix for Win95 and NT 4.0 users. Other versions will follow during the next weeks. More Information can be found at:

  • IE 3.0 security update (Microsoft)
  • Cybersnot Industries: Internet Explorer Bug (Cybersnot)
  • IE 3 security hole (C-Net)
  • IE flaw opens security risk for computer users (InfoWorld)

New Internet Explorer 3.0 security whole found
A new, different Internet Explorer security whole has been discovered. The whole may allow hackers to remotely retrieve files or install a Virus. It is not yet said, if this only applies to IE 3.0 for NT and/or Win95. More Information can be found at:

  • New IE security whole found (CNET)
