Download instructions








Download Greenland Masks screen saver:

The Greenlands Masks screen saver runs under Windows 3.1, 3.11 and Windows 95

The screen saver requires the VBRUN300.DLL file. If you already have this file on your PC, download MASKS.ZIP. If you don't have the VBRUN300.DLL file on your PC or if you are not sure, download MASKS-VB.ZIP

Click to download:

MASKS.ZIP (284.122 bytes)
MASKS-VB.ZIP (513.867 bytes)



Installation

Extract (unzip) the file. Then:
1. Copy GGMASKS.SCR file to c:\windows
(2. Copy VBRUN300.DLL file to c:\windows\system)
3. Start the Control panel (Main Group), double-click the Desktop icon, Select "GreenlandMasks" from the screen saver drop down menu.

The installation instruction is also included in the README.TXT


Don't know how to extract a .zip file? Download this self-extracting file (with the same content as MASKS-VB.ZIP):

MASKS-VB.EXE (529.637 bytes)

After download, create an empty directory on your harddisk (i.g. c:\masks) and copy the file to there. Double-click the file name to extract it. Now you'll find the files GGMASKS.SCR, VBRUN300.DLL and README.TXT in the same directory.
